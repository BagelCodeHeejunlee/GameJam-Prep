# 케이크 링크 프로토타입 규칙

## 확정 규칙

- 중앙 보드는 이번 테스트 스테이지에서 4×4다.
- 하단의 판 3개 중 원하는 순서로 배치하고, 3개를 모두 사용하면 다음 묶음이 등장한다.
- 판은 클릭 선택이 아닌 드래그 방식으로만 배치한다.
- 드래그 중인 판은 손가락이나 포인터보다 위에 떠서 보이며, 떠 있는 판이 가리키는 칸에 놓인다.
- 조각 이동은 결과를 즉시 바꾸지 않는다. 주는 판의 조각과 받는 판의 자리가 서로 마주보도록 판이 회전한 뒤, 같은 판에서 같은 색으로 이동하는 조각들은 한 묶음으로 동시에 건너간다.
- 중앙에 놓은 판은 플레이어가 다시 직접 옮길 수 없다.
- 판 하나에는 조각이 최대 6개 올라간다.
- 판을 놓으면 새로 놓은 판과 바로 상하좌우에 붙은 판까지만 한 번 정렬한다. 인접 판이 조각을 받아도 그 판의 상하좌우까지 추가로 당겨오지는 않는다.
- 일치하지 않는 색의 조각은 함께 따라오지 않고 원래 판에 그대로 남는다. 예: 기존 `A2 B1` 옆에 새 `A3`을 놓으면 `A2`만 이동해 새 판이 `A5`가 되고, 기존 판에는 `B1`이 남는다.
- 새 판에 빈자리가 하나라도 있으면 중심 색은 새 판에서 같은 색 6개가 될 때까지 인접 판의 같은 색을 모두 가져온다. 이때 자리가 부족하면 새 판의 다른 색을 인접 판으로 밀어낸다. 예: 기존 `A4` 옆에 새 `A1 B3`을 놓으면 결과는 기존 판 `B3`, 새 판 `A5`가 된다.
- 혼합 조각으로 6칸이 꽉 찬 판은 해당 이동으로 같은 색 6개를 즉시 완성할 수 있을 때만 다른 조각을 밀어내며 받는다.
- 한 번 어떤 색을 받은 판은 같은 정렬 과정에서 그 색을 다시 다른 판에 빼앗기지 않는다. 같은 두 판 사이의 왕복과 순환 이동을 금지한다.
- 가능한 종류가 여러 개면 새 판과 주변 판에 있는 같은 색 조각의 전체 합계가 가장 큰 색을 중심 색으로 정한다.
- 중심 색을 가져온 뒤 새 판에 남은 다른 색은 주변에 같은 색 판과 빈자리가 있으면 그 판으로 이동해 정렬된다.
- 자리가 부족해 밀려난 조각은 주변의 같은 종류 판으로 먼저 이동한다.
- 같은 종류 판이 없다면, 해당 조각을 밀어낸 조각이 출발한 판의 빈자리로 이동한다.
- 모든 이동이 끝난 후 같은 종류 6개인 판을 제거한다.
- 밀려난 조각을 받은 바로 인접 판이 같은 종류 6개가 되면 함께 완성 처리한다.

## 프로토타입 임시 규칙

- 게임을 열면 현재 스테이지 하나만 강조된 화면이 나타난다. 스테이지 번호, 난이도 배지, 플레이 버튼을 표시한다.
- 난이도 배지는 고정 문구가 아니라 현재 스테이지 설정의 `difficulty` 값을 읽으며, 스테이지별로 `쉬움`, `보통`, `어려움`을 지정할 수 있다.
- 메인 화면에서 게임 방법과 사운드를 조절할 수 있고, 플레이 중 홈 버튼을 누르면 새 게임 상태로 메인 화면에 돌아간다.
- 조각이 모두 빠져 빈 판이 되면 해당 판도 정리한다.
- 판 교체는 게임당 3회 가능하다.
- 같은 조건의 우선순위는 상 → 우 → 하 → 좌다.
- 스테이지 클리어 조건은 아직 적용하지 않고, 보드가 가득 찰 때까지 완성 수와 점수를 기록한다.
- 판 공급은 동일 조건을 반복 테스트할 수 있도록 고정 시드와 초반 고정 묶음을 사용한다.
- 게임 화면은 스크롤 없이 한 화면에 고정되며, 별도의 스테이지 정보 패널은 표시하지 않는다.
