# 자동 공격 디펜스 로그라이크 기획서

## 1. 피벗 목표

기존 헥스 전투와 전투 중 카드 선택/순서 선택은 이번 프로토타입의 중심에서 제외한다. 기존 메인 게임의 카드 데이터는 삭제하지 않고 보존한다. 필요하면 기존 방식으로 되돌아갈 수 있게 루트 `app.js`의 카드 정의와 보상 풀은 건드리지 않는다.

새 목표는 **3명의 영웅이 자동으로 싸우고, 유저는 웨이브 사이 선택지로 빌드를 만드는 세로형 디펜스 로그라이크**다.

핵심 재미:

> 내가 고른 업그레이드 조합이 전투에서 자동으로 굴러가며 적을 막아내는 것을 보는 재미.

## 2. 유지하는 것

- 영웅 3명이 함께 전투에 들어간다.
- 궁수/전사/마법사의 기존 성장 방향은 유지한다.
- 기존 카드/패시브 데이터는 삭제하지 않는다.
- 캐릭터별 빌드 정체성은 카드 대신 자동 행동 업그레이드로 이전한다.

## 3. 제거하는 것

전투 중에는 아래 선택을 하지 않는다.

- 카드 드로우
- 카드 순서 선택
- 공격 대상 선택
- 열/칸 선택
- 우선도
- 캐릭터 이동 명령

전투는 자동으로 진행되고, 유저는 결과를 관찰한다.

## 4. 전투 구조

- 보드: 모바일 세로형 `5열 x 7행`
- 적은 위에서 등장해 아래 성문으로 내려온다.
- 영웅 3명은 하단에서 자동 행동한다.
- 적이 성문에 닿으면 성문 체력이 감소한다.
- 웨이브가 끝나면 업그레이드 선택지가 나온다.

전투 루프:

1. 적 등장
2. 영웅 자동 공격
3. 적 이동
4. 함정/룬/화상/운석 등 자동 효과 처리
5. 웨이브 클리어 시 선택지 3개 중 1개 선택

## 5. 영웅 자동 행동

### 궁수

기존 방향: `함정`, `연타`, `표식/저격`

기본 행동:

- 위험도가 높은 적을 자동 사격한다.
- 일정 주기로 가장 위험한 줄에 함정을 설치한다.

성장 방향:

- 연타 횟수 증가
- 표식 부여
- 함정 피해/개수/연쇄 강화
- 표식 적 우선 공격

### 전사

기존 방향: `광전`, `돌진`, `범위 공격`

기본 행동:

- 성문에 가장 가까운 적을 공격한다.
- 일정 주기로 가장 가까운 적을 위로 밀어낸다.

성장 방향:

- 체력이 낮을수록 피해 증가
- 돌진 쿨다운 감소
- 밀침 강화
- 행/주변 범위 공격
- 성문 근처 적 제압

### 마법사

기존 방향: `연쇄`, `룬`, `운석`

기본 행동:

- 적이 많이 모인 줄에 마력탄을 쏜다.
- 상태이상 적에게 추가 전이를 만든다.

성장 방향:

- 연쇄 횟수 증가
- 룬 설치/폭발
- 운석 주기/피해 강화
- 상태이상 적 추가 피해
- 광역 피해 확장

## 6. 선택지 원칙

전투 중 선택은 없고, 웨이브 사이 선택지가 핵심이다.

선택지는 세 종류다.

1. 영웅 단독 강화
2. 특정 빌드 트리 강화
3. 영웅 간 시너지 강화

선택지 문장은 짧아야 한다. 화면에는 `연타 +1`, `함정 피해 +2`, `운석 주기 -20%`처럼 바로 읽히는 배지를 우선 노출한다.

## 7. 캐릭터별 선택지

### 궁수 선택지

| 이름 | 트리 | 효과 |
| --- | --- | --- |
| 빠른 사격 | 공용 | 궁수 공격 속도 증가 |
| 약점 표식 | 표식 | 궁수 공격이 표식을 부여 |
| 표식 사냥 | 표식 | 표식 적에게 추가 피해 |
| 연속 사격 | 연타 | 궁수 공격이 추가 탄을 발사 |
| 반복 리듬 | 연타 | 같은 적을 연속 공격하면 피해 증가 |
| 삼연사 | 연타 | 추가 탄 수 증가 |
| 공격 함정 | 함정 | 위험한 줄에 함정 설치 |
| 함정 손질 | 함정 | 함정 피해 증가 |
| 가시 길목 | 함정 | 함정이 적을 느리게 함 |
| 연쇄 함정 | 함정 | 함정 발동 시 주변 피해 |

### 전사 선택지

| 이름 | 트리 | 효과 |
| --- | --- | --- |
| 강한 베기 | 공용 | 전사 기본 피해 증가 |
| 방패 밀치기 | 돌진 | 주기적으로 적을 위로 밀침 |
| 돌파 | 돌진 | 밀침 피해 증가 |
| 전장의 혜성 | 돌진 | 돌진 쿨다운 감소 |
| 넓은 휩쓸기 | 범위 공격 | 전사 공격이 같은 행 적에게 피해 |
| 전열 붕괴 | 범위 공격 | 밀린 적 주변에도 피해 |
| 전장 장악 | 범위 공격 | 범위 공격 피해 증가 |
| 피의 박자 | 광전 | 성문 피해를 받을수록 전사 피해 증가 |
| 광전 태세 | 광전 | 낮은 성문 체력에서 전사 공격 속도 증가 |
| 피의 폭풍 | 광전 | 일정 시간마다 다중 타격 |

### 마법사 선택지

| 이름 | 트리 | 효과 |
| --- | --- | --- |
| 안정된 마력 | 공용 | 마법사 기본 피해 증가 |
| 연쇄 마력탄 | 연쇄 | 마법 공격이 다른 적에게 전이 |
| 세 갈래 번개 | 연쇄 | 전이 횟수 증가 |
| 무한 회로 | 연쇄 | 전이 피해 증가 |
| 폭발 룬 | 룬 | 위험한 줄에 룬 설치 |
| 룬 조율 | 룬 | 룬 피해 증가 |
| 룬 연동 | 룬 | 룬이 발동하면 주변 룬도 발동 |
| 운석 예고 | 운석 | 주기적으로 운석 낙하 |
| 파편 회수 | 운석 | 운석 처치 시 다음 운석 쿨다운 감소 |
| 하늘 붕괴 | 운석 | 운석 피해와 범위 증가 |

## 8. 파티 시너지 선택지

| 이름 | 조건/방향 | 효과 |
| --- | --- | --- |
| 표식 돌파 | 궁수 + 전사 | 전사가 표식 적을 밀면 추가 피해 |
| 표식 전도 | 궁수 + 마법사 | 마법 연쇄가 표식을 복사 |
| 불타는 함정 | 궁수 + 마법사 | 함정이 맞은 적에게 화상 부여 |
| 충돌 폭발 | 전사 + 마법사 | 밀린 적에게 마법 폭발 |
| 전선 사냥 | 궁수 + 전사 | 성문 가까운 표식 적에게 궁수 추가 사격 |
| 룬 방패 | 전사 + 마법사 | 전사 밀침 지점에 룬 생성 |
| 완성된 포위 | 3인 | 표식/밀침/마법 피해가 한 적에게 겹치면 폭발 |

## 9. 프로토타입 성공 기준

1. 전투 중 선택이 없어도 지루하지 않은가?
2. 업그레이드 선택이 전투 결과에서 눈에 보이는가?
3. 3명의 영웅이 모두 성장하는 느낌이 나는가?
4. 궁수/전사/마법사의 기존 트리 방향이 살아 있는가?
5. 카드 시스템 없이도 로그라이크 선택의 성취감이 생기는가?
