# 전사 캐릭터 기획서

작성일: 2026-07-04

## 1. 캐릭터 요약

전사는 현재 조준 방향의 가까운 위협을 매우 짧은 범위에서 강하게 끊어내는 근거리 강타 캐릭터다. 궁수가 화면상 이전 전사 사거리처럼 보이는 범위에서 적을 먼저 줄이고, 마법사가 적 무리를 지연 폭발로 정리한다면, 전사는 더 안쪽까지 들어온 핵심 적을 빠르게 끊어내는 마지막 방어선이다.

핵심 정체성:

> 짧은 사거리, 넓은 베기 각도, 높은 단일 피해, 타워 근처 추가 피해, 제한적 밀침 성장.

전사는 조합 시너지 없이도 혼자 빌드가 성립해야 한다. 캐릭터 수가 많아질 것을 고려해 특정 캐릭터와의 전용 조합 효과는 만들지 않는다.

## 2. 전투 역할

| 항목 | 내용 |
| --- | --- |
| 역할 | 근거리 강타/타워 방어 |
| 강점 | 타워에 붙은 적 처리, 가까운 고위험 적 빠른 처치, 밀침으로 짧은 시간 벌기 |
| 약점 | 사거리가 짧고 기본 타격 수가 적어, 적이 충분히 접근하기 전까지는 영향력이 제한됨 |
| 권장 배치 판단 | 적이 타워 근처까지 접근했거나, 체력 높은 적이 타워를 때리기 직전인 방향에 맞춘다 |

## 3. 기본 전투 스펙

첫 밸런스 기준값이다. 실제 수치는 전투 템포에 맞춰 조정한다.

| 항목 | 값 |
| --- | --- |
| 체력 또는 생존 규칙 | 없음. 현재 게임 구조에서는 캐릭터가 직접 공격받지 않음 |
| 공격력 | 34 |
| 공격 사거리 | 58 |
| 공격 각도 | 95도 |
| 공격 속도 | 0.92초마다 1회 공격 |
| 타겟 우선순위 | 가까운 적 우선 |
| 공격 판정 방식 | 즉발 부채꼴 베기 |
| 투사체 개수 | 해당 없음 |
| 투사체 속도 | 해당 없음 |
| 투사체 유도 여부 | 해당 없음 |
| 관통 여부 | 해당 없음. 범위 안 적을 직접 판정 |
| 범위 피해 | 있음. 기본 1명 타격. 업그레이드 후 다수 타격 가능 |
| 상태이상 | 기본 없음 |
| 설치/소환 지속시간 | 해당 없음 |
| 최대 설치/소환 개수 | 해당 없음 |
| 로테이션 관련 효과 | 기본 없음. 업그레이드로 확장 |

타겟 우선순위의 `가까운 적`은 전사의 현재 공격 범위 안에서 타워에 가장 가까운 적을 의미한다.

## 4. 기본 공격 방식

전사는 현재 조준 방향을 기준으로 짧은 부채꼴 안을 즉시 베어낸다. 공격 범위 안에 적이 있으면 타워에 가장 가까운 적 1명에게 큰 피해를 준다.

베기 규칙:

- 베기는 즉발 판정이다.
- 기본 공격은 투사체를 사용하지 않는다.
- 기본 공격은 적을 밀어내지 않는다.
- 업그레이드로 타격 수, 타워 근처 추가 피해, 밀침이 붙을 수 있다.

## 5. 런 중 업그레이드 구조

전사는 캐릭터 공통 규칙인 `성장 선택지 3개 + 기본 선택지 3개 + 돌파 선택지 1개 + 궁극 선택지 1개` 구조를 사용한다.

기본 원칙:

- 성장 선택지 3종은 1회씩만 획득 가능하며, 3종을 모두 모아야 돌파 선택지가 등장 가능하다.
- 기본 선택지 3종은 돌파 조건에 영향이 없고, 카드별 최대 랭크까지 중복 획득 가능하다.
- 성장 선택지는 타격 수 증가, 밀침, 타워 방어 추가 피해처럼 전사의 공격 방식이 달라지는 효과로 구성한다.
- 기본 선택지는 공격력, 공격 간격, 사거리처럼 스탯형 강화로 구성한다.
- 캐릭터 메타 레벨은 핵심 카드를 잠그지 않는다.
- 메타 성장은 궁극 해금, 선택지 품질, 등장 가중치에 연결한다.
- 조합 시너지 카드는 만들지 않는다.

## 6. 성장 선택지 3개

성장 선택지는 전사의 돌파 조건을 여는 핵심 카드다. 각 카드는 런 중 1회만 획득 가능하다.

| 카드명 | 효과 |
| --- | --- |
| 성벽 베기 | 타워를 공격 중인 적에게 주는 전사 피해 증가 |
| 방패 밀치기 | 전사 기본 공격이 적을 바깥으로 밀어냄 |
| 연속 베기 | 전사 기본 공격 타격 수 +1 |

성장 선택지 3개를 모두 모으면 `돌파: 철벽 돌진`이 등장 가능해진다.

## 7. 기본 선택지 3개

기본 선택지는 전사의 수치를 반복적으로 다듬는 랭크형 카드다. 기본 선택지는 돌파 조건에 영향을 주지 않는다.

| 카드명 | 최대 중복 | 효과 |
| --- | ---: | --- |
| 강철 완력 | 5회 | 전사 공격력 +5 |
| 검술 호흡 | 4회 | 전사 공격 간격 -6%, 최소 0.62초 |
| 짧은 전진 | 3회 | 전사 사거리 +4 |

## 8. 돌파 선택지 1개

돌파 선택지는 전사의 성장 선택지 3종을 모두 모으면 등장한다. 전사는 돌파까지 기본으로 가진다.

| 카드명 | 효과 |
| --- | --- |
| 돌파: 철벽 돌진 | 전사의 기본 베기가 `돌진 베기`로 변한다. 공격할 때 현재 조준 방향으로 전방 충격파를 내보내고, 범위 안 적에게 피해를 준 뒤 타워 바깥 방향으로 밀어낸다. |

의도:

- 전사의 핵심 성장인 `고화력 베기`와 `짧은 밀침`을 한눈에 보이는 공격 패턴으로 만든다.
- 돌파 이후 전사는 타워에 붙은 적을 단순히 때리는 캐릭터가 아니라, 가까운 적을 강하게 끊고 잠깐 밀어내는 캐릭터가 된다.
- 사거리보다 타격 수, 밀침, 타워 근처 추가 피해가 강화되기 때문에 궁수와 역할이 겹치지 않는다.

시각 연출:

- 돌파 획득 순간 전사 주변에 붉은 방패 문양이 떠오른다.
- 기본 베기 이펙트가 짧은 검흔에서 굵은 충격파로 바뀐다.
- 돌진 베기 방향으로 바닥 균열과 먼지 파동이 생긴다.
- 밀린 적 뒤로 짧은 충격 잔상이 남는다.
- 전사 캐릭터 아이콘/전투 모델에 붉은 어깨 갑주 오라가 유지된다.

## 9. 궁극 선택지 1개

궁극 선택지는 장기적으로 캐릭터 레벨 또는 별 레벨을 올려 해금한다. 현재 프로토타입에서는 `돌파 완료 + 전사 기본 선택지 1회 이상 + 전사 총 선택지 5회 이상 + 런 레벨 6 이상`일 때 등장 가능하다.

| 카드명 | 효과 |
| --- | --- |
| 궁극: 성벽 분쇄 | 전사가 일정 횟수 공격할 때마다 현재 조준 방향의 타워 근처에서 거대한 방패 충격파를 일으킨다. 충격파는 타워를 공격 중인 모든 적에게 큰 피해를 주고, 살아남은 적을 바깥으로 밀어낸다. |

의도:

- 전사의 최고 성장 보상은 `타워가 뚫린 상황을 되돌리는 힘`이어야 한다.
- 전사는 궁극을 통해 단순 딜러가 아니라 위기 복구형 방어 캐릭터로 완성된다.
- 현재 조준 방향과 타워 근처에 한정해 발동하므로 화면 전체 방어기가 되지 않는다.

시각 연출:

- 발동 직전 타워 근처 바닥에 거대한 방패 문양이 새겨진다.
- 0.35초 후 방패 문양이 폭발하며 붉은 충격파가 바깥으로 퍼진다.
- 맞은 적은 뒤로 밀리며 방패 파편 이펙트를 남긴다.
- 타워가 잠깐 붉은 보호막으로 감싸진다.
- 궁극 발동 시 낮은 타격음, 짧은 화면 흔들림, 카메라 펄스를 사용한다.

효과 제한:

- 현재 조준 방향 안의 타워 근처 적에게만 적용한다.
- 화면 전체 적을 밀어내지 않는다.
- 밀침 거리는 보스/정예에게 감소 적용한다.

## 10. 공용 선택지 처리

현재 레벨업 선택지 풀에서는 공용 카드를 사용하지 않는다. 로테이션 직후 첫 공격 강화는 전사 선택지에서 제거한다.

전사와 맞지 않는 방향:

- 전사를 관통형 원거리 딜러처럼 바꾸는 카드
- 화면 전체 밀침처럼 로테이션 방향 판단을 무의미하게 만드는 카드
- 모든 캐릭터 공격 방식을 동일하게 만드는 순수 스탯 카드

## 11. 메타 성장 연결

전사의 메타 성장은 핵심 카드를 잠그지 않고, 궁극 해금과 선택지 품질을 늘리는 방식으로 설계한다.

### 캐릭터 레벨

| 레벨 | 해금/효과 |
| --- | --- |
| Lv.1 | 전사 사용 가능. 핵심 선택지 등장 가능 |
| Lv.5 | 전사 선택지 등장 가중치 소폭 증가 |
| Lv.10 | 전사 돌파 선택지 접근성 증가 |
| Lv.15 | 궁극: 성벽 분쇄 해금 후보 |
| Lv.20 | 런 시작 시 전사 선택지 후보 1회 등장 |

### 별 레벨

| 별 | 해금/효과 |
| --- | --- |
| 1성 | 전사 사용 가능 |
| 2성 | 타워를 공격 중인 적에게 주는 기본 피해 소폭 증가 |
| 3성 | 돌파 선택지 품질 증가 |
| 4성 | 밀침/타워 방어 관련 카드 등장 가중치 증가 |
| 5성 | 궁극: 성벽 분쇄 해금 또는 시작 카드 선택권 |

5성 효과는 과금/수집 메타와 직결되므로, `시작 카드 선택권`이 과하면 궁극 해금만 제공하고 시작 보너스는 낮춘다.

## 12. 밸런스 리스크

| 리스크 | 설명 | 대응 |
| --- | --- | --- |
| 밀침이 너무 강함 | 적이 타워에 거의 닿지 못할 수 있음 | 밀침 거리 제한, 보스/정예 밀침 감소 |
| 궁수와 역할이 겹침 | 사거리가 길어지면 원거리 딜러처럼 느껴질 수 있음 | 투사체/관통 없음, 기본 타격 수 제한, 타워 근처 추가 피해로 역할 분리 |
| 궁극이 화면 전체 방어기가 됨 | 방향 판단이 약해질 수 있음 | 현재 조준 방향과 타워 근처로 제한 |
| 수호자와 역할 중복 | 넓은 각도 방어 캐릭터와 겹칠 수 있음 | 전사는 타워 근처/밀침, 수호자는 안정 범위 방어로 분리 |

## 13. 출시 제품 기준 판단

전사는 많은 캐릭터가 추가되어도 유지 가능한 단순한 정체성을 가져야 한다.

- 조합 시너지는 만들지 않는다.
- 핵심 카드는 처음부터 보여준다.
- 메타 성장은 궁극 해금과 선택지 품질에 연결한다.
- 런 중에는 성장 선택지 3종을 모아 돌파를 열고, 기본 선택지로 반복 성장 체감을 만든다.
- 전사의 재미는 `위험 방향 대응 -> 타워에 붙은 적 강타 -> 밀침/충격파로 짧게 전선 복구`에 있다.
