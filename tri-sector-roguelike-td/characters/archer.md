# 궁수 캐릭터 기획서

작성일: 2026-07-04

## 1. 캐릭터 요약

궁수는 긴 사거리와 빠른 투사체로 멀리서 접근하는 적을 먼저 줄이는 원거리 딜러다. 기본 캐릭터가 넓은 120도 범위를 안정적으로 방어한다면, 궁수는 더 좁은 각도와 더 긴 사거리로 정확한 방향 판단을 요구한다.

핵심 정체성:

> 좁은 공격각, 긴 사거리, 빠른 투사체, 관통 성장.

궁수는 조합 시너지 없이도 혼자 빌드가 성립해야 한다. 캐릭터 수가 많아질 것을 고려해 특정 캐릭터와의 전용 조합 효과는 만들지 않는다.

## 2. 전투 역할

| 항목 | 내용 |
| --- | --- |
| 역할 | 장거리 단일/관통 딜러 |
| 강점 | 먼 적 선제 처리, 빠른 적 대응, 직선 관통 성장 |
| 약점 | 공격각이 좁아 로테이션 방향이 틀리면 딜 손실이 큼 |
| 권장 배치 판단 | 빠른 적이나 체력 높은 적이 몰려오는 방향에 맞춘다 |

## 3. 기본 전투 스펙

첫 밸런스 기준값이다. 실제 수치는 전투 템포에 맞춰 조정한다.

| 항목 | 값 |
| --- | --- |
| 체력 또는 생존 규칙 | 없음. 현재 게임 구조에서는 캐릭터가 직접 공격받지 않음 |
| 공격력 | 26 |
| 공격 사거리 | 320 |
| 공격 각도 | 55도 |
| 공격 속도 | 0.82초마다 1회 공격 |
| 타겟 우선순위 | 가까운 적 우선 |
| 공격 판정 방식 | 직선 투사체 |
| 투사체 개수 | 1 |
| 투사체 속도 | 빠름 |
| 투사체 유도 여부 | 없음 |
| 관통 여부 | 기본 없음 |
| 범위 피해 | 없음 |
| 상태이상 | 없음 |
| 설치/소환 지속시간 | 해당 없음 |
| 최대 설치/소환 개수 | 해당 없음 |
| 로테이션 관련 효과 | 기본 없음. 업그레이드로 확장 |

타겟 우선순위의 `가까운 적`은 궁수의 현재 담당 120도 구역과 사거리 안에서 타워에 가장 가까운 적을 의미한다. 궁수의 55도 공격각은 적 탐색 조건이 아니라 화살 판정 폭, 연사 퍼짐, 궁극 범위 효율에 사용한다.

## 4. 기본 공격 방식

궁수는 현재 담당 120도 구역 안의 적을 탐색한다. 담당 구역과 사거리 안에 적이 있으면 타워에 가장 가까운 적을 향해 화살을 발사한다.

화살 규칙:

- 화살은 직선으로 빠르게 이동한다.
- 기본 화살은 유도하지 않는다.
- 기본 화살은 적 1명에게 적중하면 사라진다.
- 업그레이드로 관통, 추가 투사체, 치명타, 로테이션 직후 강화가 붙을 수 있다.

## 5. 런 중 업그레이드 구조

궁수는 캐릭터 공통 규칙인 `기본 선택지 3개 + 돌파 선택지 1개 + 궁극 선택지 1개` 구조를 사용한다.

기본 원칙:

- 핵심 카드인 사거리, 관통, 공격 속도, 치명타 계열은 Lv.1부터 등장 가능하다.
- 캐릭터 메타 레벨은 핵심 카드를 잠그지 않는다.
- 메타 성장은 궁극 해금, 선택지 품질, 등장 가중치에 연결한다.
- 조합 시너지 카드는 만들지 않는다.

## 6. 기본 선택지 3개

기본 선택지는 궁수 정체성을 만드는 핵심 카드다. 궁수를 사용하는 런에서는 처음부터 등장 가능하다.

| 카드명 | 등급 | 효과 |
| --- | --- | --- |
| 장궁 숙련 | 기본 | 사거리 증가 |
| 빠른 장전 | 기본 | 공격 속도 증가 |
| 관통 화살 | 기본 | 화살이 적 1명을 관통 |

기본 선택지 3개만으로도 궁수의 핵심 재미가 보여야 한다.

## 7. 돌파 선택지 1개

돌파 선택지는 궁수를 런 중 충분히 강화하면 등장한다. 궁수는 돌파까지 기본으로 가진다.

| 카드명 | 효과 |
| --- | --- |
| 돌파: 천공 연사 | 궁수의 기본 화살이 `천공 화살`로 변한다. 매 공격마다 부채꼴 안의 가까운 적을 향해 3발을 빠르게 연사하고, 각 화살은 적을 1회 관통한다. |

의도:

- 궁수의 두 핵심 성장인 `관통`과 `연속 사격`을 하나의 눈에 띄는 공격 패턴으로 묶는다.
- 돌파 이후 궁수는 단발 저격수가 아니라 빠르게 화살을 쏟아내는 관통 딜러로 보인다.
- 공격각을 넓히지 않고 강해지기 때문에 수호자와 역할이 겹치지 않는다.

시각 연출:

- 돌파 획득 순간 궁수 주변에 푸른 원형 파동이 퍼진다.
- 기본 화살 색이 노란색에서 푸른빛이 섞인 `천공 화살`로 바뀐다.
- 3연사 화살은 미세하게 다른 궤적으로 날아가며 잔상을 남긴다.
- 관통이 발생한 적 뒤로 짧은 빛의 궤적이 이어진다.
- 궁수 캐릭터 아이콘/전투 모델에 작은 푸른 활시위 오라가 유지된다.

## 8. 궁극 선택지 1개

궁극 선택지는 캐릭터 레벨 또는 별 레벨을 올려 해금한다. 궁극은 궁수의 장기 성장 목표다.

| 카드명 | 효과 |
| --- | --- |
| 궁극: 별비 사격 | 궁수가 일정 횟수 공격할 때마다 현재 조준 방향 전체에 별빛 표식을 찍고, 잠시 후 하늘에서 거대한 화살비가 쏟아진다. 화살비는 범위 안의 모든 적에게 피해를 주고, 타워를 공격 중인 적에게는 추가 피해를 준다. |

의도:

- 궁수의 약점인 다수 대응력을 장기 성장으로 크게 보완한다.
- 유도 투사체처럼 방향 판단을 없애지 않고, 현재 조준 방향 안에서만 발동한다.
- 캐릭터 레벨/별 레벨을 올렸을 때 체감되는 고유 보상으로 사용한다.

시각 연출:

- 발동 직전 현재 조준 방향 전체가 별빛으로 잠깐 밝아진다.
- 범위 안 적 머리 위에 작은 표식이 찍힌다.
- 0.4초 후 하늘에서 굵은 화살 여러 발이 사선으로 떨어진다.
- 화면이 짧게 흔들리고, 타격 지점마다 별가루와 파편이 튄다.
- 궁극 발동 시 일반 공격과 구분되는 전용 사운드/카메라 펄스를 사용한다.

효과 제한:

- 현재 조준 방향 안에 있는 적에게만 적용한다.
- 자동 유도처럼 화면 전체를 커버하지 않는다.
- 발동 주기와 대상 수를 제한해 광역 전문 캐릭터의 역할을 침범하지 않게 한다.

## 9. 공용 카드와의 관계

궁수는 공용 카드에 과하게 의존하지 않아야 한다. 공용 카드는 궁수의 정체성을 대체하지 않고 보조해야 한다.

궁수와 잘 맞는 공용 카드:

- 로테이션 직후 첫 공격 강화
- 타워에 붙은 적 추가 피해
- 투사체 속도 증가
- 경험치 획득량 증가
- 공격 범위 표시 강화 또는 명중 보조

궁수와 맞지 않는 공용 카드:

- 공격각을 지나치게 넓혀 수호자와 역할이 겹치는 카드
- 자동 유도 강화로 방향 판단을 무의미하게 만드는 카드
- 모든 캐릭터 공격 방식을 동일하게 만드는 순수 스탯 카드

## 10. 메타 성장 연결

궁수의 메타 성장은 핵심 카드를 잠그지 않고, 궁극 해금과 선택지 품질을 늘리는 방식으로 설계한다.

### 캐릭터 레벨

| 레벨 | 해금/효과 |
| --- | --- |
| Lv.1 | 궁수 사용 가능. 핵심 일반 카드 등장 가능 |
| Lv.5 | 궁수 기본 선택지 등장 가중치 소폭 증가 |
| Lv.10 | 궁수 돌파 선택지 접근성 증가 |
| Lv.15 | 궁극: 별비 사격 해금 후보 |
| Lv.20 | 런 시작 시 궁수 기본 선택지 후보 1회 등장 |

### 별 레벨

| 별 | 해금/효과 |
| --- | --- |
| 1성 | 궁수 사용 가능 |
| 2성 | 기본 사거리 소폭 증가 |
| 3성 | 돌파 선택지 품질 증가 |
| 4성 | 치명타/관통 관련 카드 등장 가중치 증가 |
| 5성 | 궁극: 별비 사격 해금 또는 시작 카드 선택권 |

5성 효과는 과금/수집 메타와 직결되므로, `시작 카드 선택권`이 과하면 궁극 해금만 제공하고 시작 보너스는 낮춘다.

## 11. 밸런스 리스크

| 리스크 | 설명 | 대응 |
| --- | --- | --- |
| 좁은 각도 때문에 답답함 | 방향이 틀리면 공격을 못 한다 | 긴 사거리와 높은 피해로 보상 |
| 관통 빌드가 과하게 강함 | 일렬 적 배치에서 모든 적을 삭제할 수 있음 | 관통 후 피해 감소 또는 관통 수 제한 |
| 유도 카드가 방향 판단을 죽임 | 로테이션 게임의 핵심을 약화시킴 | 기본 유도 없음, 유도는 제한적/희귀 효과로만 검토 |
| 수호자와 역할 중복 | 공격각 증가를 과하게 주면 넓은 범위 딜러가 됨 | 궁수 공격각 증가는 제한적으로 운영 |
| 궁극의 광역 성능 과다 | 별비 사격이 수호자/광역 캐릭터 역할을 침범할 수 있음 | 현재 조준 방향 안에서만 발동, 발동 주기/대상 수 제한 |

## 12. 출시 제품 기준 판단

궁수는 많은 캐릭터가 추가되어도 유지 가능한 단순한 정체성을 가져야 한다.

- 조합 시너지는 만들지 않는다.
- 핵심 카드는 처음부터 보여준다.
- 메타 성장은 궁극 해금과 선택지 품질에 연결한다.
- 런 중에는 기본 선택지를 쌓아 돌파를 열어 성장 체감을 만든다.
- 궁수의 재미는 `정확한 방향 선택 -> 먼 적 선제 제거 -> 관통/연속 사격으로 성장`에 있다.
