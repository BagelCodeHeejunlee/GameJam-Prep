# 궁수 캐릭터 기획서

작성일: 2026-07-04

## 1. 캐릭터 요약

궁수는 긴 사거리와 빠른 투사체로 멀리서 접근하는 적을 먼저 줄이는 원거리 딜러다. 현재 테스트 기준에서는 모든 캐릭터의 기본 공격각을 120도로 통일하고, 궁수는 더 긴 사거리와 직선 투사체/관통 성장으로 차별화한다.

핵심 정체성:

> 120도 공격각, 긴 사거리, 빠른 투사체, 관통 성장.

궁수는 조합 시너지 없이도 혼자 빌드가 성립해야 한다. 캐릭터 수가 많아질 것을 고려해 특정 캐릭터와의 전용 조합 효과는 만들지 않는다.

## 2. 전투 역할

| 항목 | 내용 |
| --- | --- |
| 역할 | 장거리 단일/관통 딜러 |
| 강점 | 먼 적 선제 처리, 빠른 적 대응, 직선 관통 성장 |
| 약점 | 투사체가 직선이라 타겟이 빗나가거나 과잉 화력이 낭비될 수 있음 |
| 권장 배치 판단 | 빠른 적이나 체력 높은 적이 몰려오는 방향에 맞춘다 |

## 3. 기본 전투 스펙

첫 밸런스 기준값이다. 실제 수치는 전투 템포에 맞춰 조정한다.

| 항목 | 값 |
| --- | --- |
| 체력 또는 생존 규칙 | 없음. 현재 게임 구조에서는 캐릭터가 직접 공격받지 않음 |
| 공격력 | 26 |
| 공격 사거리 | 135 |
| 공격 각도 | 120도 |
| 공격 속도 | 0.82초마다 1회 공격 |
| 타겟 우선순위 | 가까운 적 우선 |
| 공격 판정 방식 | 직선 투사체 |
| 투사체 개수 | 1 |
| 투사체 속도 | 빠름 |
| 투사체 유도 여부 | 없음 |
| 관통 여부 | 기본 없음 |
| 범위 피해 | 없음 |
| 상태이상 | 없음 |
| 설치/소환 지속시간 | 해당 없음 |
| 최대 설치/소환 개수 | 해당 없음 |
| 로테이션 관련 효과 | 기본 없음. 업그레이드로 확장 |

타겟 우선순위의 `가까운 적`은 궁수의 현재 공격 범위 안에서 타워에 가장 가까운 적을 의미한다.

## 4. 기본 공격 방식

궁수는 현재 조준 방향을 기준으로 120도 부채꼴 안의 적을 탐색한다. 공격 범위 안에 적이 있으면 타워에 가장 가까운 적에게 화살을 발사한다.

화살 규칙:

- 화살은 직선으로 빠르게 이동한다.
- 기본 화살은 유도하지 않는다.
- 기본 화살은 적 1명에게 적중하면 사라진다.
- 업그레이드로 추가 투사체, 관통, 폭발 화살이 붙을 수 있다.
- 추가 투사체는 부채꼴로 퍼지지 않고 같은 대상에게 발사된다. 시각적으로는 발사 위치와 타이밍이 나뉘며, 각 화살은 같은 타겟 지점으로 수렴한다.

## 5. 런 중 업그레이드 구조

궁수는 캐릭터 공통 규칙인 `성장 선택지 3개 + 기본 선택지 3개 + 돌파 선택지 1개 + 고급 선택지 3개 + 궁극 선택지 1개` 구조를 사용한다.

기본 원칙:

- 성장 선택지 3종은 1회씩만 획득 가능하며, 3종을 모두 모아야 돌파 선택지가 등장 가능하다.
- 기본 선택지 3종은 돌파 조건에 영향이 없고, 카드별 최대 랭크까지 중복 획득 가능하다.
- 성장 선택지는 투사체 수 증가, 관통, 폭발 화살처럼 궁수의 공격 방식이 달라지는 효과로 구성한다.
- 기본 선택지는 공격력, 공격 간격, 사거리처럼 스탯형 강화로 구성한다.
- 고급 선택지는 돌파 이후에만 등장하며, 천공 포화를 더 강하게 확장하는 1회성 카드로 구성한다.
- 캐릭터 메타 레벨은 핵심 카드를 잠그지 않는다.
- 메타 성장은 궁극 해금, 선택지 품질, 등장 가중치에 연결한다.
- 조합 시너지 카드는 만들지 않는다.

## 6. 성장 선택지 3개

성장 선택지는 궁수의 돌파 조건을 여는 핵심 카드다. 각 카드는 런 중 1회만 획득 가능하다.

| 카드명 | 효과 |
| --- | --- |
| 연발 시위 | 궁수가 같은 대상에게 화살을 1발 더 발사 |
| 관통 시위 | 궁수 화살 관통 +1 |
| 폭렬 촉 | 궁수 화살 적중 시 주변 적에게 작은 폭발 피해 |

성장 선택지 3개를 모두 모으면 `돌파: 천공 포화`가 등장 가능해진다.

## 7. 기본 선택지 3개

기본 선택지는 궁수의 수치를 반복적으로 다듬는 랭크형 카드다. 기본 선택지는 돌파 조건에 영향을 주지 않는다.

| 카드명 | 최대 중복 | 효과 |
| --- | ---: | --- |
| 정밀 조준 | 5회 | 궁수 공격력 +5 |
| 빠른 장전 | 4회 | 궁수 공격 간격 -8%, 최소 0.42초 |
| 장궁 숙련 | 4회 | 궁수 사거리 +12 |

## 8. 돌파 선택지 1개

돌파 선택지는 궁수의 성장 선택지 3종을 모두 모으면 등장한다. 궁수는 돌파까지 기본으로 가진다.

| 카드명 | 효과 |
| --- | --- |
| 돌파: 천공 포화 | 궁수의 기본 화살이 `천공 화살`로 변한다. 같은 대상 다중 사격, 관통, 폭발 효과가 한 단계 강화되어 더 많은 화살이 적을 뚫고 지나가며 주변 적에게 폭발 피해를 남긴다. |

의도:

- 궁수의 세 핵심 성장인 `같은 대상 다중 사격`, `관통`, `폭발`을 하나의 눈에 띄는 공격 패턴으로 묶는다.
- 돌파 이후 궁수는 단발 저격수가 아니라 빠르게 화살을 쏟아내는 관통 딜러로 보인다.
- 공격각을 더 넓히지 않고 투사체 성능으로 강해지기 때문에 안정 범위 딜러와 역할이 겹치지 않는다.

시각 연출:

- 돌파 획득 순간 궁수 주변에 푸른 원형 파동이 퍼진다.
- 기본 화살 색이 노란색에서 푸른빛이 섞인 `천공 화살`로 바뀐다.
- 3연사 화살은 다른 발사 위치와 짧은 시간차를 두고 같은 목표로 수렴하며 잔상을 남긴다.
- 관통이 발생한 적 뒤로 짧은 빛의 궤적이 이어지고, 적중 지점에는 작은 폭발 파동이 남는다.
- 궁수 캐릭터 아이콘/전투 모델에 작은 푸른 활시위 오라가 유지된다.

## 9. 고급 선택지 3개

고급 선택지는 `돌파: 천공 포화` 이후에 등장한다. 각 카드는 런 중 1회만 획득 가능하다.

| 카드명 | 효과 |
| --- | --- |
| 천공 분열 | 같은 대상에게 발사하는 궁수 화살 +1, 화살 피해 소폭 증가 |
| 유성 관통 | 궁수 화살 관통 +1, 관통 화살 피해 증가 |
| 성운 폭렬 | 폭발 화살 반경과 폭발 피해 증가 |

의도:

- 천공 포화의 세 축인 `연발`, `관통`, `폭발` 중 하나를 한 번 더 밀어준다.
- 고급 선택지는 순수 스탯이 아니라 돌파 이후 공격 모양이 더 과장되어 보이게 만든다.
- 공격각은 120도 기준을 유지하고, 궁수의 정체성을 장거리 직선 화력으로 유지한다.

## 10. 궁극 선택지 1개

궁극 선택지는 장기적으로 캐릭터 레벨 또는 별 레벨을 올려 해금한다. 현재 프로토타입에서는 `돌파 완료 + 궁수 기본 선택지 1회 이상 + 궁수 총 선택지 5회 이상 + 런 레벨 6 이상`일 때 등장 가능하다.

| 카드명 | 효과 |
| --- | --- |
| 궁극: 별비 사격 | 궁수가 일정 횟수 공격할 때마다 현재 조준 방향 전체에 별빛 표식을 찍고, 잠시 후 하늘에서 거대한 화살비가 쏟아진다. 화살비는 범위 안의 모든 적에게 피해를 주고, 타워를 공격 중인 적에게는 추가 피해를 준다. |

의도:

- 궁수의 약점인 다수 대응력을 장기 성장으로 크게 보완한다.
- 유도 투사체처럼 방향 판단을 없애지 않고, 현재 조준 방향 안에서만 발동한다.
- 캐릭터 레벨/별 레벨을 올렸을 때 체감되는 고유 보상으로 사용한다.

시각 연출:

- 발동 직전 현재 조준 방향 전체가 별빛으로 잠깐 밝아진다.
- 범위 안 적 머리 위에 작은 표식이 찍힌다.
- 0.4초 후 하늘에서 굵은 화살 여러 발이 사선으로 떨어진다.
- 화면이 짧게 흔들리고, 타격 지점마다 별가루와 파편이 튄다.
- 궁극 발동 시 일반 공격과 구분되는 전용 사운드/카메라 펄스를 사용한다.

효과 제한:

- 현재 조준 방향 안에 있는 적에게만 적용한다.
- 자동 유도처럼 화면 전체를 커버하지 않는다.
- 발동 주기와 대상 수를 제한해 광역 전문 캐릭터의 역할을 침범하지 않게 한다.

## 11. 공용 선택지 처리

현재 레벨업 선택지 풀에서는 공용 카드를 사용하지 않는다. 로테이션 직후 첫 공격 강화는 궁수 선택지에서 제거한다.

궁수와 맞지 않는 방향:

- 공격각을 120도보다 더 넓혀 안정 광역 캐릭터와 역할이 겹치는 카드
- 자동 유도 강화로 방향 판단을 무의미하게 만드는 카드
- 모든 캐릭터 공격 방식을 동일하게 만드는 순수 스탯 카드

## 12. 메타 성장 연결

궁수의 메타 성장은 핵심 카드를 잠그지 않고, 궁극 해금과 선택지 품질을 늘리는 방식으로 설계한다.

### 캐릭터 레벨

| 레벨 | 해금/효과 |
| --- | --- |
| Lv.1 | 궁수 사용 가능. 핵심 선택지 등장 가능 |
| Lv.5 | 궁수 선택지 등장 가중치 소폭 증가 |
| Lv.10 | 궁수 돌파 선택지 접근성 증가 |
| Lv.15 | 궁극: 별비 사격 해금 후보 |
| Lv.20 | 런 시작 시 궁수 선택지 후보 1회 등장 |

### 별 레벨

| 별 | 해금/효과 |
| --- | --- |
| 1성 | 궁수 사용 가능 |
| 2성 | 기본 사거리 소폭 증가 |
| 3성 | 돌파 선택지 품질 증가 |
| 4성 | 치명타/관통 관련 카드 등장 가중치 증가 |
| 5성 | 궁극: 별비 사격 해금 또는 시작 카드 선택권 |

5성 효과는 과금/수집 메타와 직결되므로, `시작 카드 선택권`이 과하면 궁극 해금만 제공하고 시작 보너스는 낮춘다.

## 13. 밸런스 리스크

| 리스크 | 설명 | 대응 |
| --- | --- | --- |
| 120도 통일로 정체성이 약해짐 | 공격각 차별화가 사라지면 다른 딜러와 비슷해질 수 있음 | 긴 사거리, 빠른 직선 투사체, 관통/연발 성장으로 분리 |
| 관통 빌드가 과하게 강함 | 일렬 적 배치에서 모든 적을 삭제할 수 있음 | 관통 후 피해 감소 또는 관통 수 제한 |
| 유도 카드가 방향 판단을 죽임 | 로테이션 게임의 핵심을 약화시킴 | 기본 유도 없음, 유도는 제한적/희귀 효과로만 검토 |
| 수호자와 역할 중복 | 공격각을 더 넓히면 안정 범위 딜러가 됨 | 궁수 공격각 증가는 보류 |
| 궁극의 광역 성능 과다 | 별비 사격이 수호자/광역 캐릭터 역할을 침범할 수 있음 | 현재 조준 방향 안에서만 발동, 발동 주기/대상 수 제한 |

## 14. 출시 제품 기준 판단

궁수는 많은 캐릭터가 추가되어도 유지 가능한 단순한 정체성을 가져야 한다.

- 조합 시너지는 만들지 않는다.
- 핵심 카드는 처음부터 보여준다.
- 메타 성장은 궁극 해금과 선택지 품질에 연결한다.
- 런 중에는 성장 선택지 3종을 모아 돌파를 열고, 기본 선택지로 반복 성장 체감을 만든다.
- 궁수의 재미는 `정확한 방향 선택 -> 먼 적 선제 제거 -> 관통/연속 사격으로 성장`에 있다.
