# 마법사 캐릭터 기획서

작성일: 2026-07-04

## 1. 캐릭터 요약

마법사는 중거리에서 적이 몰린 지점에 마법진을 설치하고, 잠시 뒤 폭발시키는 지연 광역 딜러다. 궁수가 먼 적을 정확히 먼저 줄이고, 전사가 타워 근처 위기를 밀어낸다면, 마법사는 적이 뭉친 방향을 예측해서 한 번에 정리하는 역할을 맡는다.

핵심 정체성:

> 중거리, 지연 폭발, 작은 광역, 잔류 장판 성장.

마법사는 조합 시너지 없이도 혼자 빌드가 성립해야 한다. 캐릭터 수가 많아질 것을 고려해 특정 캐릭터와의 전용 조합 효과는 만들지 않는다.

## 2. 전투 역할

| 항목 | 내용 |
| --- | --- |
| 역할 | 중거리 광역/지연 피해 |
| 강점 | 몰려오는 잡몹 처리, 뭉친 적 대응, 잔류 장판으로 지속 압박 |
| 약점 | 공격 발동이 느리고, 빠른 적이나 단일 고체력 적 처리 효율이 낮음 |
| 권장 배치 판단 | 적이 한 방향에 뭉치기 시작했거나, 곧 타워에 접근할 무리를 미리 줄이고 싶을 때 맞춘다 |

타겟 우선순위의 `가까운 적`은 마법사의 현재 공격 범위 안에서 타워에 가장 가까운 적을 의미한다.

## 4. 기본 공격 방식

마법사는 현재 조준 방향을 기준으로 부채꼴 안의 적을 탐색한다. 공격 범위 안에 적이 있으면 타워에 가장 가까운 적의 현재 위치에 작은 마법진을 생성하고, 0.35초 뒤 폭발시킨다.

마법진 규칙:

- 마법진은 시전 시점의 적 위치에 고정된다.
- 기본 마법진은 적을 추적하지 않는다.
- 폭발 반경 안의 모든 적에게 피해를 준다.
- 기본 마법진은 폭발 후 장판을 남기지 않는다.
- 이미 생성된 마법진은 로테이션이 시작되어도 폭발까지 처리된다. 단, 로테이션 중에는 새 공격을 하지 않는다.
- 업그레이드로 잔류 장판, 연쇄 폭발, 2차 폭발, 폭발 반경, 공격 속도가 붙을 수 있다.

## 5. 런 중 업그레이드 구조

마법사는 캐릭터 공통 규칙인 `성장 선택지 3개 + 기본 선택지 3개 + 돌파 선택지 1개 + 궁극 선택지 1개` 구조를 사용한다.

기본 원칙:

- 성장 선택지 3종은 1회씩만 획득 가능하며, 3종을 모두 모아야 돌파 선택지가 등장 가능하다.
- 기본 선택지 3종은 돌파 조건에 영향이 없고, 카드별 최대 랭크까지 중복 획득 가능하다.
- 성장 선택지는 잔류 장판, 연쇄 마법진, 2차 폭발처럼 마법진의 작동 방식이 달라지는 효과로 구성한다.
- 기본 선택지는 공격력, 폭발 반경, 공격 간격처럼 스탯형 강화로 구성한다.
- 캐릭터 메타 레벨은 핵심 카드를 잠그지 않는다.
- 메타 성장은 궁극 해금, 선택지 품질, 등장 가중치에 연결한다.
- 조합 시너지 카드는 만들지 않는다.

## 6. 성장 선택지 3개

성장 선택지는 마법사의 돌파 조건을 여는 핵심 카드다. 각 카드는 런 중 1회만 획득 가능하다.

| 카드명 | 효과 |
| --- | --- |
| 잔류 마력 | 마법진 폭발 후 짧은 지속 피해 장판 생성 |
| 연쇄 룬 | 마법진 폭발 후 가까운 적 1명에게 보조 마법진 생성 |
| 메아리 룬 | 마법진 폭발 후 같은 위치에 작은 2차 폭발 |

성장 선택지 3개를 모두 모으면 `돌파: 균열 마법진`이 등장 가능해진다.

## 7. 기본 선택지 3개

기본 선택지는 마법사의 수치를 반복적으로 다듬는 랭크형 카드다. 기본 선택지는 돌파 조건에 영향을 주지 않는다.

| 카드명 | 최대 중복 | 효과 |
| --- | ---: | --- |
| 응축 마력 | 5회 | 마법사 공격력 +3 |
| 확산 마법진 | 4회 | 마법사 폭발 반경 +6 |
| 빠른 영창 | 4회 | 마법사 공격 간격 -7%, 최소 0.55초 |

## 8. 돌파 선택지 1개

돌파 선택지는 마법사의 성장 선택지 3종을 모두 모으면 등장한다. 마법사는 돌파까지 기본으로 가진다.

| 카드명 | 효과 |
| --- | --- |
| 돌파: 균열 마법진 | 마법사의 기본 마법진이 `균열 마법진`으로 변한다. 폭발 후 현재 조준 방향 안의 가까운 적 최대 2명에게 작은 보조 마법진이 연쇄 생성되고, 짧은 지연 후 다시 폭발한다. |

의도:

- 마법사의 핵심 성장인 `지연 폭발`과 `광역 연쇄`를 한눈에 보이는 공격 패턴으로 만든다.
- 돌파 이후 마법사는 단일 지점 폭발 캐릭터가 아니라, 몰려오는 적 무리를 차례로 터뜨리는 광역 캐스터가 된다.
- 현재 조준 방향 안에서만 연쇄되므로 로테이션 판단을 없애지 않는다.

시각 연출:

- 돌파 획득 순간 마법사 주변에 금색 룬 원이 겹겹이 회전한다.
- 기본 마법진 색이 작은 원형 문양에서 균열이 있는 큰 룬 문양으로 바뀐다.
- 첫 폭발 이후 보조 마법진까지 얇은 빛의 선이 연결된다.
- 보조 마법진은 짧게 맥동한 뒤 폭발한다.
- 마법사 캐릭터 아이콘/전투 모델에 떠다니는 작은 룬 조각 오라가 유지된다.

## 9. 궁극 선택지 1개

궁극 선택지는 장기적으로 캐릭터 레벨 또는 별 레벨을 올려 해금한다. 현재 프로토타입에서는 `돌파 완료 + 마법사 기본 선택지 1회 이상 + 마법사 총 선택지 5회 이상 + 런 레벨 6 이상`일 때 등장 가능하다.

| 카드명 | 효과 |
| --- | --- |
| 궁극: 대마법진 붕괴 | 마법사가 일정 횟수 공격할 때마다 현재 조준 방향의 중거리 지점에 거대한 대마법진을 펼친다. 0.6초 후 대마법진이 붕괴하며 큰 피해를 주고, 잠시 동안 균열 장판을 남겨 범위 안 적에게 지속 피해를 준다. |

의도:

- 마법사의 최고 성장 보상은 `적이 몰린 방향을 예측해서 크게 터뜨리는 힘`이어야 한다.
- 궁극은 즉시 화면 전체를 지우는 기술이 아니라, 조준 방향과 지연 시간을 읽었을 때 큰 보상을 주는 기술이어야 한다.
- 잔류 장판까지 포함해 마법사의 장기 빌드 방향인 `공간 장악`을 완성한다.

시각 연출:

- 발동 직전 현재 조준 방향 바닥에 거대한 룬 원이 천천히 펼쳐진다.
- 룬 원 안쪽으로 작은 문양들이 빨려 들어가며 붕괴 예고를 준다.
- 0.6초 후 중심부가 밝게 터지고, 바닥에 갈라진 균열 장판이 남는다.
- 장판 안의 적에게 작은 마력 파편이 지속적으로 튄다.
- 궁극 발동 시 깊은 저음, 짧은 화면 흔들림, 카메라 펄스를 사용한다.

효과 제한:

- 현재 조준 방향 안의 중거리 범위에만 적용한다.
- 화면 전체 적을 자동으로 타격하지 않는다.
- 장판 지속시간과 동시 존재 수를 제한해 설치형 캐릭터의 역할을 침범하지 않게 한다.

## 10. 공용 선택지 처리

현재 레벨업 선택지 풀에서는 공용 카드를 사용하지 않는다. 장판, 연쇄, 2차 폭발은 마법사 전용 성장으로, 공격력/반경/공격 간격은 기본 카드로 배치한다.

마법사와 맞지 않는 방향:

- 마법진이 화면 전체 적을 자동 추적하게 만드는 카드
- 둔화/정지 효과를 과하게 늘려 로테이션 판단을 무의미하게 만드는 카드
- 모든 캐릭터 공격 방식을 동일하게 만드는 순수 스탯 카드
- 장판을 무제한 중첩하게 만드는 카드

## 11. 메타 성장 연결

마법사의 메타 성장은 핵심 카드를 잠그지 않고, 궁극 해금과 선택지 품질을 늘리는 방식으로 설계한다.

### 캐릭터 레벨

| 레벨 | 해금/효과 |
| --- | --- |
| Lv.1 | 마법사 사용 가능. 핵심 선택지 등장 가능 |
| Lv.5 | 마법사 선택지 등장 가중치 소폭 증가 |
| Lv.10 | 마법사 돌파 선택지 접근성 증가 |
| Lv.15 | 궁극: 대마법진 붕괴 해금 후보 |
| Lv.20 | 런 시작 시 마법사 선택지 후보 1회 등장 |

### 별 레벨

| 별 | 해금/효과 |
| --- | --- |
| 1성 | 마법사 사용 가능 |
| 2성 | 기본 폭발 반경 소폭 증가 |
| 3성 | 돌파 선택지 품질 증가 |
| 4성 | 잔류 장판/광역 관련 카드 등장 가중치 증가 |
| 5성 | 궁극: 대마법진 붕괴 해금 또는 시작 카드 선택권 |

5성 효과는 과금/수집 메타와 직결되므로, `시작 카드 선택권`이 과하면 궁극 해금만 제공하고 시작 보너스는 낮춘다.

## 12. 밸런스 리스크

| 리스크 | 설명 | 대응 |
| --- | --- | --- |
| 지연 공격이 답답함 | 빠른 적이 폭발 전에 범위를 벗어날 수 있음 | 기본 반경을 너무 작게 잡지 않고, 시각 예고를 명확히 제공 |
| 광역 성능이 과함 | 적이 뭉치는 웨이브에서 혼자 대부분을 처리할 수 있음 | 폭발 반경, 연쇄 대상 수, 장판 지속시간 제한 |

| 수호자와 역할 중복 | 공격각이 넓어지면 안정 범위 딜러가 될 수 있음 | 마법사는 지연 광역과 장판, 수호자는 안정 즉발 방어로 분리 |

## 13. 출시 제품 기준 판단

마법사는 많은 캐릭터가 추가되어도 유지 가능한 단순한 정체성을 가져야 한다.

- 조합 시너지는 만들지 않는다.
- 핵심 카드는 처음부터 보여준다.
- 메타 성장은 궁극 해금과 선택지 품질에 연결한다.
- 런 중에는 성장 선택지 3종을 모아 돌파를 열고, 기본 선택지로 반복 성장 체감을 만든다.
- 마법사의 재미는 `적 무리 예측 -> 마법진 설치 -> 지연 폭발/잔류 장판으로 공간 장악`에 있다.
