# 공병 캐릭터 기획서

작성일: 2026-07-06

## 1. 캐릭터 요약

공병은 현재 조준 방향에 지뢰를 미리 설치하고, 적이 감지 반경에 들어오면 폭발시키는 설치형 폭발 캐릭터다.

핵심 정체성:

> 즉시 공격이 아니라, 회전으로 위험 방향을 먼저 잡고 멈춰서 방어선을 깔아두는 캐릭터.

공병의 목적은 회전 조작에 `지금 맞히기` 외의 이유를 추가하는 것이다. 궁수/저격수는 조준선을 맞추는 캐릭터고, 마법사는 적 위치에 지연 폭발을 찍는 캐릭터라면, 공병은 적이 지나갈 방향을 예측해 공간을 먼저 점유한다.

## 2. 전투 역할

| 항목 | 내용 |
| --- | --- |
| 역할 | 설치형 폭발 / 경로 선점 |
| 강점 | 한 방향으로 몰려오는 무리, 러너, 탱커 진입 전 사전 처리 |
| 약점 | 회전 중 설치 불가, 설치 후 무장 시간이 있어 즉시 대응이 약함 |
| 권장 배치 판단 | 적이 곧 지나갈 방향을 예상하거나, 타워 앞 접근로를 미리 막고 싶을 때 맞춘다 |

## 4. 기본 공격 방식

- 공병은 회전 중에는 새 지뢰를 설치하지 않는다.
- 현재 조준 방향 안에 적이 있으면, 타워에 가장 가까운 적의 진행 경로 앞쪽에 지뢰를 설치한다.
- 조준 방향 안에 적이 없으면, 현재 조준 방향의 중거리 지점에 지뢰를 미리 설치한다.
- 지뢰는 짧은 무장 시간 후 활성화된다.
- 활성화된 지뢰는 적이 감지 반경에 들어오면 폭발하고, 폭발 반경 안의 모든 적에게 피해를 준다.
- 이미 설치된 지뢰는 로테이션 중에도 유지되고 기폭된다.

## 5. 런 중 업그레이드 구조

공병은 캐릭터 공통 규칙인 `성장 선택지 3개 + 기본 선택지 3개 + 돌파 선택지 1개 + 고급 선택지 3개 + 궁극 선택지 1개` 구조를 사용한다.

기본 원칙:

- 성장 선택지는 지뢰의 작동 방식이 달라지는 효과로 구성한다.
- 기본 선택지는 피해, 설치 간격, 폭발 반경 같은 수치형 강화로 구성한다.
- 돌파는 분산 매설, 감속, 재기폭을 하나의 지뢰망 패턴으로 통합한다.
- 고급 선택지는 돌파 이후 지뢰망의 밀도와 안정성을 높인다.
- 궁극은 현재 조준 방향에 대형 지뢰밭을 펼치는 형태로, 방향 판단 보상을 크게 만든다.

## 6. 성장 선택지 3개

| 카드명 | 효과 |
| --- | --- |
| 분산 매설 | 한 번에 설치하는 지뢰 +1 |
| 점착 폭약 | 지뢰 폭발에 맞은 적이 짧게 감속 |
| 예비 기폭장치 | 지뢰가 한 번 더 기폭 가능 |

성장 선택지 3개를 모두 모으면 `돌파: 지뢰망 구축`이 등장 가능해진다.

## 7. 기본 선택지 3개

| 카드명 | 최대 중복 | 효과 |
| --- | ---: | --- |
| 고폭 장약 | 5회 | 공병 지뢰 피해 +4 |
| 빠른 매설 | 4회 | 공병 설치 간격 -7%, 최소 0.72초 |
| 폭약 반경 | 4회 | 공병 지뢰 폭발 반경 +5 |

## 8. 돌파 선택지 1개

| 카드명 | 효과 |
| --- | --- |
| 돌파: 지뢰망 구축 | 분산 매설, 감속, 재기폭이 강화된 지뢰망으로 전환된다. 설치 수, 감지 반경, 폭발 반경, 최대 유지 지뢰 수가 증가한다. |

의도:

- 공병의 세 성장 축을 `한 방향을 미리 막는 방어선`으로 통합한다.
- 돌파 이후 공병은 단발 폭발 캐릭터가 아니라, 조준 방향에 짧은 지뢰망을 구축하는 캐릭터가 된다.
- 화면 전체 자동 타격이 아니라 현재 조준 방향에만 힘을 집중한다.

## 9. 고급 선택지 3개

| 카드명 | 효과 |
| --- | --- |
| 격자 매설 | 설치 지뢰 +1, 최대 유지 지뢰 증가 |
| 감지 신관 | 지뢰 감지 반경과 폭발 반경 증가 |
| 잔류 기폭 | 지뢰가 지속시간 종료 시 자동 폭발 |

## 10. 궁극 선택지 1개

| 카드명 | 효과 |
| --- | --- |
| 궁극: 폭쇄 지대 | 일정 설치마다 현재 조준 방향에 대형 지뢰밭을 전개한다. 지뢰밭은 짧게 무장된 뒤 적이 밟거나 시간이 끝나면 폭발한다. |

효과 제한:

- 현재 조준 방향에만 지뢰밭을 펼친다.
- 적을 자동 추적하지 않는다.
- 설치 후 무장 시간이 있어 회전과 예측 판단을 계속 요구한다.

## 11. 밸런스 리스크

| 리스크 | 설명 | 대응 |
| --- | --- | --- |
| 자동 방어가 과해짐 | 지뢰가 너무 오래 남거나 너무 많이 깔리면 회전 조작 이유가 줄어듦 | 최대 유지 지뢰 수, 지속시간, 무장 시간을 제한 |
| 즉시 대응이 약함 | 적이 이미 타워에 붙은 상황에서는 설치형 공격이 늦을 수 있음 | 전사/마법사와 역할을 나누고, 돌파 후 감지 반경을 보강 |
| 화면 가독성 저하 | 지뢰와 폭발 반경이 많아지면 전장이 지저분해질 수 있음 | 작은 지뢰 본체, 낮은 알파의 반경 표시, 궁극만 강한 연출 사용 |

## 12. 출시 제품 기준 판단

공병은 네 번째 영웅부터 시작되는 `조합 선택`을 검증하기 위한 캐릭터다.

- 공병이 재미있으면 이후 설치형, 소환형, 경로 제어형 캐릭터를 확장할 수 있다.
- 공병이 너무 자동적이면 지뢰 지속시간/최대 수/무장 시간을 줄여 회전 판단을 다시 요구해야 한다.
- 공병이 약하게 느껴지면 적 웨이브를 좁은 경로 무리형으로 구성해 설치형의 장점이 보이게 해야 한다.
