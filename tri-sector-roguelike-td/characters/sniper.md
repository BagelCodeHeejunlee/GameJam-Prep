# 저격수 캐릭터 기획

## 1. 캐릭터 요약

| 항목 | 내용 |
| --- | --- |
| 이름 | 저격수 |
| 역할 | 15도 초장거리 정밀 딜러 |
| 전투 정체성 | 좁은 각도를 정확히 맞췄을 때 긴 사거리와 강한 한 방으로 고위험 적을 먼저 줄이는 캐릭터 |
| 현재 프로토타입 목적 | 연속 회전 조작이 좁은 공격각 캐릭터를 살릴 수 있는지 검증 |
| 타겟 우선순위 | 현재 조준 방향 안의 가까운 적 우선 |

## 2. 전투 역할

저격수는 넓은 범위를 안정적으로 방어하는 캐릭터가 아니다. 15도의 매우 좁은 공격각을 정확히 맞췄을 때, 긴 사거리로 정예/탱커에게 강한 한 방을 꽂는 역할을 맡는다.

저격수의 재미는 `맞추기 어렵고 느리지만 맞추면 강하다`에 있다. 연속 회전 방식에서 유저가 각도를 미세하게 조정할 이유를 만드는 실험 캐릭터다.

## 3. 기본 전투 스펙

| 항목 | 값 |
| --- | ---: |
| 체력 | 없음 |
| 공격력 | 68 |
| 공격각 | 15도 |
| 사거리 | 166 |
| 공격 간격 | 1.55초 |
| 투사체 방식 | 직선 고속 탄환 |
| 투사체 수 | 1 |
| 투사체 속도 | 760 |
| 투사체 판정 폭 | 5 |
| 관통 | 1 |
| 유도 여부 | 없음 |
| 궁극 발동 주기 | 공격 7회마다 |

## 4. 기본 공격 방식

- 저격수는 현재 조준 방향 기준 15도 안의 적을 타겟팅한다.
- 타겟은 기존 공통 규칙대로 가까운 적 우선이다.
- 탄환은 매우 빠르게 직선으로 날아가며, 기본 1회 관통한다.
- 공격 간격이 길기 때문에 방향을 맞춘 뒤 한 방을 넣는 리듬을 가진다.
- 회전 중에는 공격하지 않는다.

## 5. 런 중 업그레이드 구조

저격수는 캐릭터 공통 규칙인 `성장 선택지 3개 + 기본 선택지 3개 + 돌파 선택지 1개 + 고급 선택지 3개 + 궁극 선택지 1개` 구조를 사용한다.

기본 원칙:

- 성장 선택지는 관통, 탄환 판정 폭, 과충전처럼 저격수의 공격 방식과 템포가 달라지는 효과로 구성한다.
- 기본 선택지는 공격력, 공격 간격, 사거리처럼 스탯형 강화로 구성한다.
- 고급 선택지는 돌파 이후에만 등장하며, 정밀 저격의 강점을 더 밀어주는 1회성 카드로 구성한다.
- 공격각은 기본적으로 15도를 유지하고, 차별화는 긴 사거리/느린 고화력/관통으로 만든다.

## 6. 성장 선택지 3개

| 카드명 | 효과 |
| --- | --- |
| 철갑탄 | 저격수 탄환 관통 +1 |
| 대구경탄 | 저격수 탄환 판정 폭 증가 |
| 과충전 사격 | 저격수 공격력 크게 증가, 공격 간격 증가 |

성장 선택지 3개를 모두 모으면 `돌파: 사선 개방`이 등장 가능해진다.

## 7. 기본 선택지 3개

| 카드명 | 최대 중복 | 효과 |
| --- | ---: | --- |
| 영점 조정 | 5회 | 저격수 공격력 +10 |
| 호흡 정리 | 4회 | 저격수 공격 간격 -5%, 최소 1.05초 |
| 장총열 | 4회 | 저격수 사거리 +14 |

## 8. 돌파 선택지 1개

| 카드명 | 효과 |
| --- | --- |
| 돌파: 사선 개방 | 저격수 공격이 관통, 피해, 탄속, 탄환 판정 폭이 강화된 정밀 저격으로 전환된다. |

의도:

- 좁은 각도를 정확히 맞춘 보상을 크게 만든다.
- 돌파 이후에도 광역 캐릭터가 아니라 정밀 관통 딜러로 남는다.
- 회전 조작의 세밀한 조준감이 사라지지 않도록 공격각 증가는 주지 않는다.

## 9. 고급 선택지 3개

| 카드명 | 효과 |
| --- | --- |
| 관통선 확장 | 저격수 탄환 관통 +1, 피해 증가 |
| 중심선 고정 | 저격수 탄환 폭과 피해 증가 |
| 초장거리 탄 | 저격수 사거리, 탄속, 피해 증가 |

## 10. 궁극 선택지 1개

| 카드명 | 효과 |
| --- | --- |
| 궁극: 궤도 저격 | 일정 공격마다 현재 조준 방향에 얇고 긴 저격선을 발사한다. 저격선에 닿은 적은 큰 피해를 받는다. |

효과 제한:

- 현재 조준 방향의 얇은 직선 범위에만 적용한다.
- 화면 전체 자동 타격으로 확장하지 않는다.
- 현재 조준 방향을 맞추는 조작 재미를 유지한다.

## 11. 밸런스 리스크

| 리스크 | 설명 | 대응 |
| --- | --- | --- |
| 너무 어려움 | 15도 공격각 때문에 적을 거의 못 때릴 수 있음 | 긴 사거리, 높은 피해, 관통으로 맞췄을 때의 보상을 크게 유지 |
| 너무 강함 | 회전 조작에 익숙해지면 정예를 과도하게 빨리 제거할 수 있음 | 공격 간격과 광역 부재로 무리 대응 약점 유지 |
| 연속 회전 피로 | 계속 미세 조준해야 하면 손 피로가 커질 수 있음 | 회전 속도/감속, 공격각 표시, 웨이브 밀도 조정으로 대응 |

## 12. 출시 제품 기준 판단

저격수는 연속 회전 방식이 성공했는지 판단하기 위한 기준 캐릭터다.

- 저격수가 재미있으면 좁은 공격각을 가진 고난도 캐릭터를 더 만들 수 있다.
- 저격수가 과하면 공격 간격, 피해량, 관통 수, 웨이브 구성을 다시 조정한다.
- 저격수가 너무 어렵다면 15도 캐릭터는 기본 지급이 아니라 후반 해금/상급자용으로 분리한다.
