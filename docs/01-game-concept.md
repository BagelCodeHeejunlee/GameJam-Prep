# 게임 콘셉트

## 한 줄 설명

플레이어가 캐릭터를 선택해 여러 웨이브로 구성된 스테이지에 들어가고, 헥사 타일 맵 위에서 카드 기반 턴 행동으로 전투가 자동 진행되는 RPG.

## 플레이어 판타지

플레이어는 캐릭터를 준비해 스테이지에 진입하고, 덱 구성, 카드 우선권, 사거리, 이동, 상성에 따라 전투가 굴러가는 모습을 관찰하고 해석한다.

## 핵심 루프

1. 플레이어가 캐릭터를 선택한다.
2. 스테이지에 입장한다.
3. 여러 웨이브를 순서대로 전투한다.
4. 매 턴 전투 참여자들이 덱에서 카드를 뽑는다.
5. 카드의 우선권에 따라 행동을 처리한다.
6. 웨이브를 클리어하면 보상 카드 4장 중 1장을 선택한다.
7. 선택한 카드를 이번 스테이지 동안 덱에 넣고 다음 웨이브를 진행한다.
8. 스테이지 클리어 또는 실패까지 반복한다.

## 순간 플레이

- 캐릭터들은 헥사 타일로 이루어진 맵 위에서 이동한다.
- 전투는 자동으로 진행된다.
- 아군과 적은 각자의 덱에서 나온 카드에 따라 행동한다.
- 플레이어는 보드 상황, 카드 행동, 캐릭터 상성을 보고 전투 흐름을 이해한다.
- 웨이브 클리어 후 보상 카드를 선택해 캐릭터의 이번 스테이지 빌드를 만든다.

## 승리와 패배 조건

스테이지의 모든 웨이브를 클리어하면 스테이지를 클리어한다.

현재 스테이지는 15웨이브 구성으로 잡는다.

5, 10, 15웨이브는 보스 웨이브다.

모든 플레이어 캐릭터가 사망하면 패배한다.

## 초기 프로토타입 범위

- 플레이어블 캐릭터 1종.
- 적 1종.
- 작은 헥사 타일 전투 맵.
- 15개 웨이브로 구성된 스테이지 1개.
- 최소 덱/카드 시스템.
- 카드 우선권에 따른 턴 처리.
- 웨이브 클리어 시 2x2 배열의 카드 보상 선택지.

## 아직 범위 밖

- 보상 카드 등장 확률의 세부 규칙.
- 웨이브 사이 회복, 제거, 강화, 붙이기 같은 추가 보상 시스템.
