# 리스크와 추후 결정

기획상 중요하지만 지금 당장 확정하지 않고 나중에 더 디테일하게 잡을 항목을 모은다.

## 보상 카드 등장 규칙

보상 카드 등장 규칙은 나중에 별도로 디테일하게 잡는다.

현재 미정 항목:

- 보상 4장을 뽑을 때 루트별 균형을 보정할지.
- 카드 등급별 등장 확률.
- 한 번 선택한 루트에 가까운 카드가 더 자주 나오는지.
- 이미 선택한 카드와 시너지가 있는 카드를 우선 노출할지.
- 특정 등급이나 루트가 너무 늦게 나오지 않도록 보정할지.

리스크:

- 등장 규칙이 너무 랜덤하면 원하는 빌드를 만들기 어렵다.
- 등장 규칙이 너무 보정되면 매번 비슷한 빌드가 될 수 있다.
- 루트별 카드 풀이 충분하지 않으면 보상 선택지가 반복적으로 느껴질 수 있다.

## 아직 남은 전투 규칙 미정 항목

- 후퇴 중 완전히 막힌 경우 대체 위치 선택 기준.

## 웨이브 사이 추가 보상

웨이브 사이에 카드 제거, 회복, 카드 붙이기 같은 추가 보상이 있을 수 있다.

아직 어떤 방식으로 넣을지는 정하지 않았다.

검토할 수 있는 방향:

- 카드 제거.
- 체력 회복.
- 카드 강화.
- 카드에 다른 효과를 붙이기.
- 일시적인 다음 웨이브 버프.

## 보스 웨이브 특별 보상

5, 10, 15웨이브는 보스 웨이브다.

보스 웨이브 클리어 후에는 일반 카드 보상과 다른 특별 보상이 있을 수 있다.

특별 보상의 종류와 지급 타이밍은 아직 정하지 않았다.

검토할 수 있는 방향:

- 강력한 전용 카드 선택.
- 캐릭터 패시브 강화.
- 덱 압축 또는 카드 제거.
- 체력 회복 또는 최대 체력 증가.
- 다음 구간 전체에 적용되는 축복/저주 선택.
