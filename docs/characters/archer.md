# 궁수 캐릭터 기획

## 캐릭터 개요

궁수는 기본 사거리가 긴 원거리 캐릭터다.

기본 덱은 단순한 이동, 사격, 후퇴로 구성되어 있으며, 웨이브 보상 선택에 따라 다단중첩, 함정, 차지 루트로 분기한다.

## 시작 스탯

`시작`이 붙은 값은 스테이지 밖에서 캐릭터를 성장시키면 변경될 수 있는 값이다.

| 스탯 | 값 |
| --- | ---: |
| 시작 체력 | 80 |
| 시작 공격력 | 10 |
| 기본 사거리 | 2 |
| 기본 이동력 | 2 |
| 공격 속도 | 4 |
| 이동 속도 | 2 |

공격 속도와 이동 속도는 카드 우선권의 기준값이다.

값이 높을수록 우선권 숫자가 낮아져 더 빨리 발동되는 방향으로 설계한다.

궁수 카드는 공격 액션이 포함된 카드의 우선권을 전반적으로 조금 늦게 잡고, 후퇴만 있는 카드는 빠르게 발동되는 방향으로 설계한다.

궁수 카드에서 다음 액션에 공격이 남아 있으면 위치 조정은 이동으로 적고, 더 이상 공격이 남아 있지 않으면 거리 벌리기 목적의 후퇴로 적는다.

## 궁수 전용 키워드

### 다단중첩

같은 턴에 같은 적을 여러 번 공격할수록 피해가 증가하는 계열이다.

기본 보정은 카드별 효과를 따른다.

### 함정

궁수의 함정 카드는 빈 칸에 함정을 설치한다.

함정은 칸을 점유하지 않는다.

함정은 설치자, 아군, 적을 가리지 않고 누구든 밟으면 발동한다.

자동 이동 판단에서는 함정을 가능한 한 피하지만, 함정을 밟는 경로밖에 없다면 밟고 이동할 수 있다.

함정은 발동 후 사라진다.

함정 효과는 공격 2와 이동 불가 두 종류로 통일한다.

이동 불가는 다음 해당 유닛의 턴 종료 시 제거된다.

함정 설치 카드의 우선권 숫자는 전반적으로 낮게 잡아, 적 이동보다 먼저 전장에 함정을 깔 수 있게 한다.

궁수의 특수 카드 중 일부는 함정을 설치한 뒤 적을 밀어 바로 함정을 밟게 만든다.

### 차지

차지 카드는 차지 스택을 쌓는다.

차지 상태에서는 다음 공격 전까지 이동할 수 없다.

차지 스택은 최대치가 없다.

공격 액션을 실행하면 차지 스택을 자동으로 모두 소비한다.

최종 공격 배수는 `카드 공격 배수 + 차지 스택`으로 계산한다.

예를 들어 차지 스택이 3인 상태에서 공격 2를 실행하면 최종 공격 배수는 5가 된다.

공격하지 못했다면 차지 스택은 유지된다.

사용하지 않은 차지 스택은 다음 웨이브 시작 시 사라진다.

차지 스택 정보는 전투 화면 HUD에 표시한다.

함정 설치 위치는 설치 가능 위치 중 적 근처와 플레이어-적 사이에 있는 칸을 우선한다.

우선 조건을 만족하는 칸이 없다면 설치 가능 위치 중 무작위로 선택한다.

다단중첩 피해 증가는 카드별 보정과 곱연산으로 처리한다.
