# 마법사 캐릭터 기획

## 캐릭터 개요

마법사는 긴 사거리와 다중 타겟, 위치 전환으로 전장을 제어하는 원거리 캐릭터다.

기본 체력은 낮지만 기본 사거리가 길고, 보상 선택에 따라 연쇄, 룬, 공간 루트로 분기한다.

## 시작 스탯

| 스탯 | 값 |
| --- | ---: |
| 시작 체력 | 55 |
| 시작 공격력 | 12 |
| 기본 사거리 | 3 |
| 기본 이동력 | 2 |
| 공격 속도 | 5 |
| 이동 속도 | 2 |

## 빌드 루트

| 루트 | 핵심 방향 |
| --- | --- |
| 연쇄 | 여러 적을 동시에 맞히거나 반복 타격한다. |
| 룬 | 특정 타일과 패턴을 활용해 큰 피해를 만든다. |
| 공간 | 점프 이동, 후퇴, 위치 교환으로 안전 거리를 유지한다. |
