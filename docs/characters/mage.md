# 마법사 캐릭터 기획

## 캐릭터 개요

마법사는 긴 사거리, 다중 타겟, 타일 기반 마법으로 전장을 제어하는 원거리 캐릭터다.

기본 체력은 낮지만 기본 사거리가 길고, 보상 선택에 따라 연쇄, 룬, 공간 루트로 분기한다.

## 시작 스탯

| 스탯 | 값 |
| --- | ---: |
| 시작 체력 | 55 |
| 시작 공격력 | 12 |
| 기본 사거리 | 3 |
| 기본 이동력 | 2 |
| 공격 속도 | 5 |
| 이동 속도 | 2 |

마법사는 공격 우선권이 빠른 편이지만, 체력이 낮아 위치 선정 실패가 크게 위험해야 한다.

## 빌드 루트

| 루트 | 핵심 방향 |
| --- | --- |
| 연쇄 | 원거리 공격의 타겟 수와 반복 공격을 늘려 여러 적을 동시에 처리한다. |
| 룬 | 후퇴하면서 바닥에 룬을 뿌리고, 룬 주변 적을 공격하거나 룬을 폭파한다. |
| 공간 | 정확한 거리와 위치 조건을 맞추면 매우 강력한 피해를 준다. |

## 마법사 전용 키워드

### 연쇄

연쇄는 원거리 공격의 타겟 수와 반복 공격을 늘리는 루트다.

카드 설명은 루트명보다 실제 적용 대상을 우선해서 적는다.

타겟 수를 늘리는 효과는 `원거리 공격 타겟 수 +1`처럼 표기한다.

### 룬

룬은 칸을 점유하지 않는 바닥 오브젝트다.

룬 위로 이동할 수 있고, 룬 위에서 이동을 종료할 수 있다.

아군이 설치한 룬은 아군이 밟아도 발동하지 않는다.

적이 룬을 밟으면 룬이 발동해 피해를 주고 사라진다.

룬 트리는 대부분 후퇴 이동과 함께 룬을 바닥에 뿌리는 느낌으로 구성한다.

룬 카드의 주요 사용 방식:

- 후퇴하면서 룬 설치.
- 룬 위치로 후퇴하거나 룬에 가까운 위치로 후퇴.
- 룬 위 또는 룬에 인접한 적 공격.
- 룬을 제거하면서 주변을 폭파.
- 여러 룬을 깔아둔 뒤 한 번에 폭파.

### 공간

공간은 맞히기 어렵지만 성공하면 매우 강한 루트다.

핵심 조건은 정확한 거리, 특정 방향, 이동 후 위치 조건이다.

예를 들어 `정확히 4칸 떨어진 적에게 공격 4`처럼 조건을 맞추지 못하면 공격이 실패할 수 있다.

공간 루트는 안전 거리 유지 루트가 아니라, 위험한 위치 퍼즐을 맞추는 고난도 고화력 루트로 설계한다.
