# 전사 캐릭터 기획

## 캐릭터 개요

전사는 근거리에서 적을 붙잡고 버티며 전투 흐름을 만드는 캐릭터다.

기본 덱은 접근, 근거리 공격, 짧은 범위 휩쓸기로 구성된다.

웨이브 보상 선택에 따라 방패, 광전, 제압 루트로 분기한다.

## 시작 스탯

| 스탯 | 값 |
| --- | ---: |
| 시작 체력 | 100 |
| 시작 공격력 | 8 |
| 기본 사거리 | 1 |
| 기본 이동력 | 2 |
| 공격 속도 | 3 |
| 이동 속도 | 3 |

## 빌드 루트

| 루트 | 핵심 방향 |
| --- | --- |
| 방패 | 피해 감소, 반격, 버티기로 근거리 교전을 오래 유지한다. |
| 광전 | 체력이 낮거나 많이 맞을수록 공격성이 높아진다. |
| 제압 | 밀기, 기절, 위치 제어로 적의 행동과 배치를 흔든다. |
