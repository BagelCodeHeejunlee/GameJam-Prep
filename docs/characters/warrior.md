# 전사 캐릭터 기획

## 캐릭터 개요

전사는 근거리에서 적진으로 들어가 전투 흐름을 강제로 여는 캐릭터다.

기본 덱은 접근, 근거리 공격, 짧은 범위 휩쓸기로 구성된다.

웨이브 보상 선택에 따라 광전, 돌진, 범위 공격 루트로 분기한다.

## 시작 스탯

| 스탯 | 값 |
| --- | ---: |
| 시작 체력 | 100 |
| 시작 공격력 | 8 |
| 기본 사거리 | 1 |
| 기본 이동력 | 2 |
| 공격 속도 | 3 |
| 이동 속도 | 3 |

전사는 궁수보다 짧은 사거리를 가지지만, 체력과 진입 능력으로 이를 보완한다.

## 빌드 루트

| 루트 | 핵심 방향 |
| --- | --- |
| 광전 | 체력이 낮거나 많이 공격할수록 공격성이 높아진다. |
| 돌진 | 높은 이동력과 점프 이동으로 전장 위치를 선점한다. 일부 카드는 타겟 도착 후 남은 이동력을 공격으로 바꾼다. |
| 범위 공격 | 적진에 들어가 주변 여러 칸을 휩쓸고, 여러 명을 맞힐수록 피해가 증가한다. |

## 전사 전용 키워드

### 돌진

돌진은 별도의 공격 타입이 아니라 이동에 특화된 빌드 루트다.

대부분의 돌진 카드는 `이동 6`, `점프 이동 4`처럼 이동량과 이동 방식 자체가 강하다.

일부 카드만 특수하게 타겟에게 도착한 뒤 남은 이동력을 공격으로 바꾼다.

이런 카드의 기본 공식은 `최종 공격 배수 = 카드 공격 배수 + 남은 이동력`이다.

카드에 별도 값이 있으면 남은 이동력 1칸당 공격 배수 증가량을 더 크게 줄 수 있다.

예를 들어 이동 4 후 남은 이동력만큼 공격하는 카드로 1칸만 이동했다면 남은 이동력 3이 공격으로 바뀌어 최종 공격 4가 된다.

### 범위 공격

범위 공격은 전사가 적진 안으로 들어가 여러 적을 한 번에 때리는 루트다.

기존 제압형 플레이가 가지고 있던 밀기, 위치 흔들기, 적진 진입 감각은 범위 공격 루트에 흡수한다.

범위 공격 카드는 주변 휩쓸기, 붙은 두 칸 공격, 붙은 세 칸 공격, 전방 두 칸 공격처럼 여러 타일을 공격한다.

한 번에 여러 적을 맞히면 카드별로 공격 배수가 증가할 수 있다.

### 광전

광전은 체력 손실과 반복 공격을 공격성으로 바꾸는 루트다.

다단 히트, 낮은 체력 보너스, 처치 후 추가 공격 같은 효과를 중심으로 구성한다.

`광전 태세`의 피해 증가량은 `잃은 체력 비율 / 2`로 계산한다.

예를 들어 최대 체력 100인 전사가 체력 50이면 잃은 체력 비율은 50%이고, 공격 피해는 25% 증가한다.

광전 루트에는 자기 최대 체력의 일부를 깎고 강하게 공격하는 카드와, 다시 체력을 회복하는 카드가 함께 들어간다.
