# 캐릭터 덱 구성

## 플레이어 캐릭터 기본 덱

플레이어 캐릭터는 기본적으로 8장의 기본 카드를 가지고 게임에 입장한다.

보상 카드를 얻었을 때 새 카드가 더 자주 등장하게 하기 위해 시작 덱은 8장으로 잡는다.

보통 구성은 다음과 같다.

| 카드 분류 | 장수 |
| --- | ---: |
| 이동 + 공격 카드 | 3장 |
| 공격 + 후퇴 카드 | 2장 |
| 공격 카드 | 1장 |
| 후퇴 카드 | 1장 |
| 특수 공격 카드 | 1장 |
| 합계 | 8장 |

이 8장의 카드를 웨이브의 매 턴마다 뽑아서 사용한다.

전투 템포를 위해 기본 카드 대부분은 `이동 -> 공격` 또는 `공격 -> 후퇴`처럼 두 개 이상의 행동을 가진다.

순수 후퇴 카드와 순수 공격 카드는 적은 수만 유지하고, 높은 수치나 명확한 목적을 가진 카드로 사용한다.

덱이 모두 떨어지면 버린카드 더미를 다시 섞어서 덱으로 사용한다.

웨이브 시작 시에는 항상 전체 덱을 셔플해서 사용한다.

## 웨이브 보상 카드

각 캐릭터는 웨이브 클리어 보상으로 등장할 수 있는 카드 풀을 가진다.

웨이브를 클리어하면 해당 캐릭터의 보상 카드 풀에서 카드 4장을 보여준다.

현재 프로토타입에서는 이 4장을 2x2 배열로 보여주며, 그중 1장을 선택한다.

플레이어는 그중 1장을 선택해야 한다.

선택한 카드는 캐릭터의 덱에 들어오며, 이번 스테이지 동안 사용할 수 있다.

## 플레이어 덱 순환

플레이어 캐릭터의 기본 카드 흐름:

1. 덱에서 카드를 뽑는다.
2. 카드를 사용한다.
3. 사용한 기본 카드는 버린카드 더미로 이동한다.
4. 덱이 비면 버린카드 더미를 섞어 새 덱으로 만든다.

지속 카드와 영구 카드는 기본 카드와 버린카드 더미 이동 타이밍이 다르다.

- 지속 카드는 다음 내 턴 시작 전까지 효과를 유지하고, 내 턴이 시작할 때 버린카드 더미로 이동한다.
- 영구 카드는 이번 웨이브가 끝날 때까지 유지되며, 그동안 버린카드 더미로 들어가지 않는다.

웨이브 종료 처리와 다음 웨이브 시작 준비 단계에서 영구 카드와 버린카드 더미의 카드가 모두 다시 덱으로 들어온다.

## 적 덱

적들은 보통 4장에서 10장으로 구성된 자기만의 덱을 사용한다.

같은 종류의 적은 같은 적 덱에서 뽑은 카드 한 장을 공유해서 사용한다.

적 덱도 덱이 모두 떨어지면 버린카드 더미를 다시 섞어서 사용한다.

적도 지속 카드와 영구 카드를 사용할 수 있다.

예시:

- 임프 덱은 6장으로 구성된다.
- 이번 턴 임프 덱에서 카드 한 장을 뽑는다.
- 살아있는 모든 임프는 그 카드를 사용한다.

## 아직 정하지 않은 내용

- 플레이어 캐릭터가 시작 덱을 수정할 수 있는지.
- 보상으로 카드 추가 외에 카드 제거나 강화가 있는지.
