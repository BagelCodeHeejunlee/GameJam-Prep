## 현재 전투 HUD 방향

전투 화면은 모바일 세로모드 기준으로 설계한다.

보드는 화면 전체를 채우고, HUD는 보드 위에 얹힌다.

상단 HUD:

- 적 정보를 먼저 보여준다.
- 적 정보는 적 종류별 섹션으로 나눈다.
- 각 적 섹션에는 적 라벨/초상과 남은 수량을 우선 보여준다. 공간이 충분할 때만 기본 이동력, 기본 공격력, 기본 사거리를 아이콘과 숫자로 함께 보여준다.
- 적 카드는 상시 표시하지 않는다.
- 적 정보 옆에는 이번 턴 우선권 순서만 작은 토큰으로 보여준다.

하단 HUD:

- 플레이어 캐릭터가 1명이라면 같은 크기의 캐릭터 HUD를 중앙에 둔다.
- 플레이어 캐릭터가 2명이라면 좌우로 분리해서 각각의 HUD를 보여준다.
- 각 플레이어 HUD에는 캐릭터 라벨/초상, 기본 이동력, 기본 공격력, 기본 사거리, 덱 수, 버린 더미 수, 차지 수, 이번 턴 뽑은 카드 미리보기를 둔다.
- 체력은 현재 HUD에서 우선 노출하지 않는다.

카드 공개 연출:

- 전투 HUD를 숨기지 않는다.
- 별도 카드 공개 오버레이를 만들지 않고 실제 메인 우선권 HUD 안에서 진행한다.
- 우선권 HUD는 먼저 드로우 상태를 보여주고, 이후 같은 영역 안에서 정렬 상태로 전환된다.
- 정렬이 끝난 우선권 HUD는 그대로 현재 턴 실행 순서 표시 역할을 이어받는다.
- 반복되는 연출이 루즈하지 않도록 정렬 후 대기 시간은 짧게 유지한다.

## 액션 아이콘

| 항목 | 용도 | 우선순위 | 현재 상태 |
| --- | --- | --- | --- |
| 점프 이동 | 벽, 적, 장애물을 넘는 이동 카드 | 높음 | 일반 이동 아이콘으로 표시 중 |
| 후퇴 | 일반 이동과 구분되는 거리 벌리기 | 높음 | 일반 신발 아이콘 재사용 중 |
| 남은 이동 공격 | 전사 돌진 루트의 특수 공격 조건 | 높음 | 텍스트 노트로 표시 중 |
| 자해 | 광전 카드의 체력 감소 비용 | 높음 | 텍스트 노트로 표시 중 |
| 회복 | 체력 회복 카드 | 높음 | 텍스트 노트로 표시 중 |
| 광전 태세 | 잃은 체력 비율 기반 피해 증가 | 중간 | 영구 효과 텍스트로 표시 중 |
| 체력 조건 강화 | 체력 25% 미만 피해 증가 | 중간 | 영구 효과 텍스트로 표시 중 |
| 밀기 | 전사 범위 공격, 함정 연계 | 높음 | 텍스트 노트로 표시 중 |
| 당기기 | 추후 제어 카드 | 중간 | 텍스트 노트로 표시 중 |
| 차지 | 차지 스택 획득 | 중간 | 단순 + 아이콘 사용 중 |
| 방어/피해 감소 | 지속 카드와 방어 효과 | 중간 | 전용 아이콘 없음 |
| 반격 | 지속 카드 반격 효과 | 중간 | 전용 아이콘 없음 |

## 우선 제작 추천

가장 먼저 만들면 체감이 큰 항목:

1. 점프 이동, 자해, 회복, 밀기 아이콘.
2. 룬 설치, 룬 폭파, 운석 예고 아이콘.
3. 주변 1칸/2칸 범위 아이콘.
4. 영구 효과와 지속 효과 상태 아이콘.
5. 운석 낙하, 룬 폭파, 회복 이펙트.
