# 자동 방 클리어 전투 설계

## 1. 핵심 방향

이 게임의 전투는 `하데스`나 `궁수의 전설`처럼 방 단위로 진행된다. 단, 전투 중 플레이어는 직접 이동하거나 타겟을 고르지 않는다. 한 방에 들어가면 캐릭터가 자동으로 움직이고, 자동으로 적을 판단하고, 자동으로 공격한다.

플레이어가 하는 일:

- 런 시작 전 캐릭터와 무기 선택
- 방 클리어 후 증강 보드에서 선택
- 다음 방으로 진입
- 런 전체 빌드 방향 판단

플레이어가 하지 않는 일:

- 전투 중 이동 조작
- 전투 중 공격 버튼 입력
- 전투 중 스킬 버튼 입력
- 전투 중 타겟 지정
- 전투 중 회피 입력

기준 문장:

> 전투는 자동으로 감상하고, 플레이어의 실력은 전투 전에 고른 조합에서 나온다.

## 2. 방 진행 구조

한 스테이지는 여러 개의 방으로 구성된다.

기본 구조:

1. 방 진입
2. 문이 닫힘
3. 몬스터 웨이브 등장
4. 캐릭터가 자동 이동/자동 타겟팅/자동 공격
5. 방 안의 모든 몬스터 처치
6. 보상 또는 증강 보드 등장
7. 선택 후 다음 방 이동

방 안의 모든 몬스터를 처치해야 다음 방으로 넘어간다. 시간 생존형이 아니라 `방 클리어형`이다.

## 3. 웨이브 구조

한 방에는 1개 이상의 내부 웨이브가 있을 수 있다.

예:

- 일반 방: 1-2 웨이브
- 엘리트 방: 2-3 웨이브
- 보스 방: 보스 1마리 + 소환 웨이브

중요한 점은 “웨이브가 자동으로 무한히 밀려오는 생존 게임”이 아니라, 방마다 정해진 적 구성을 처리하는 구조라는 것이다.

## 4. 자동 전투 AI

전투 중 캐릭터는 아래 판단을 자동으로 수행한다.

### 자동 이동

캐릭터는 현재 무기, 캐릭터 성향, 적 위치, 위험 지역에 따라 이동한다.

이동 성향 예:

- 원거리형: 적과 거리를 유지하며 원을 그리듯 이동
- 근접형: 낮은 위험도의 적에게 접근 후 빠르게 이탈
- 소환형: 소환체 뒤쪽에 위치하며 안전거리를 유지
- 탱커형: 적을 자신에게 모으고 위험 지역을 짧게 벗어남
- 포탑형: 설치물 주변을 지키며 이동 반경이 작음

### 자동 타겟팅

타겟팅은 무기와 빌드에 따라 달라진다.

타겟팅 규칙 예:

- 가장 가까운 적
- 체력이 가장 낮은 적
- 엘리트/보스 우선
- 몰려 있는 적 중심
- 코어/소환체를 공격 중인 적 우선
- 상태이상이 걸린 적 우선
- 표식이 있는 적 우선

### 자동 스킬 사용

스킬 버튼은 없다. 스킬은 조건을 만족하면 자동 발동한다.

예:

- 쿨다운마다 자동 사용
- 적이 3마리 이상 모이면 사용
- 체력이 40% 이하가 되면 사용
- 엘리트 등장 시 사용
- 방 시작 시 사용
- 적 처치 시 연쇄 발동

## 5. 캐릭터 설계

캐릭터는 단순 스킨이 아니라 빌드의 출발점이다. 각 캐릭터는 자동 이동 성향, 기본 패시브, 선호 태그를 가진다.

### 캐릭터 공통 데이터

- 이름
- 역할
- 자동 이동 성향
- 기본 패시브
- 선호 태그
- 약점
- 추천 무기

### 캐릭터 예시

#### 레인저

역할:

- 안정적인 원거리 자동 전투

자동 이동:

- 가장 가까운 적과 일정 거리를 유지하며 원형 이동

기본 패시브:

- 같은 적을 연속 공격할수록 공격 속도 증가

선호 태그:

- `탄환`
- `상태`
- `처치`

잘 맞는 무기:

- 자동 석궁
- 리볼버
- 전기 소총

#### 블레이드 댄서

역할:

- 이동과 근접 범위 피해

자동 이동:

- 적 무리 가장자리로 접근한 뒤 대시로 빠져나감

기본 패시브:

- 이동 거리에 따라 다음 공격 범위 증가

선호 태그:

- `이동`
- `근접`
- `장판`

잘 맞는 무기:

- 쌍검
- 궤도검
- 에너지 낫

#### 엔지니어

역할:

- 포탑/드론/설치물 중심

자동 이동:

- 설치물 근처를 유지하며 적을 유도

기본 패시브:

- 방 시작 시 임시 포탑 1개 설치

선호 태그:

- `소환`
- `코어`
- `경제`

잘 맞는 무기:

- 드론 컨트롤러
- 터렛 키트
- 지뢰 발사기

#### 워든

역할:

- 방어/반격/코어 보호

자동 이동:

- 적과 코어 사이를 막는 위치를 우선함

기본 패시브:

- 피해를 받을 때 주변 적에게 반격 피해

선호 태그:

- `방어`
- `코어`
- `처치`

잘 맞는 무기:

- 방패 해머
- 충격 장갑
- 보호막 발생기

#### 아크 메이지

역할:

- 상태이상/광역/연쇄 피해

자동 이동:

- 적 밀집 지역을 피해 사거리 안에서 유지

기본 패시브:

- 상태이상 적에게 주는 피해 증가

선호 태그:

- `상태`
- `장판`
- `탄환`

잘 맞는 무기:

- 번개 지팡이
- 냉기 구체
- 화염서

## 6. 무기 설계

무기는 전투의 기본 공격 방식과 타겟팅 규칙을 결정한다. 같은 캐릭터라도 무기에 따라 빌드 방향이 달라져야 한다.

### 무기 공통 데이터

- 이름
- 사거리
- 공격 주기
- 타겟팅 규칙
- 공격 형태
- 기본 태그
- 잘 맞는 증강
- 약점

### 무기 예시

#### 자동 석궁

공격 형태:

- 가장 가까운 적에게 빠른 단일 탄환

기본 태그:

- `탄환`
- `처치`

시너지:

- 관통
- 다중 발사
- 처치 폭발
- 표식

#### 리볼버

공격 형태:

- 느리지만 강한 탄환. 체력이 낮은 적 우선

기본 태그:

- `탄환`
- `치명`

시너지:

- 처형
- 재장전 보상
- 보스 추가 피해

#### 쌍검

공격 형태:

- 근처 적에게 빠른 연속 베기

기본 태그:

- `근접`
- `이동`

시너지:

- 대시
- 궤도검
- 이동 거리 보상
- 피격 반격

#### 번개 지팡이

공격 형태:

- 랜덤 적에게 번개. 감전된 적 사이로 연쇄

기본 태그:

- `상태`
- `연쇄`

시너지:

- 감전 전이
- 상태이상 피해
- 처치 연쇄

#### 드론 컨트롤러

공격 형태:

- 드론이 독립적으로 적을 공격

기본 태그:

- `소환`
- `상태`

시너지:

- 드론 편대
- 드론 전도체
- 드론 회수
- 표식 부여

#### 터렛 키트

공격 형태:

- 방 시작 시 터렛 설치. 터렛이 자동 공격

기본 태그:

- `코어`
- `소환`

시너지:

- 포탑 속사
- 설치물 보호막
- 코어 반격

## 7. 캐릭터 + 무기 조합 예시

### 레인저 + 자동 석궁

초기 방향:

- 안정적인 탄환/처치 빌드

잘 맞는 증강:

- 관통 화살
- 쌍열 발사
- 처치 폭발
- 표식 사냥

플레이 감각:

- 화면을 가로지르는 탄환이 적을 정리한다.

### 블레이드 댄서 + 쌍검

초기 방향:

- 이동/근접/장판 빌드

잘 맞는 증강:

- 자동 대시
- 플라즈마 잔상
- 충격 파동
- 궤도검 폭풍

플레이 감각:

- 캐릭터가 방 안을 스타일리시하게 가르며 적을 베고 지나간다.

### 엔지니어 + 터렛 키트

초기 방향:

- 설치물/코어/소환 빌드

잘 맞는 증강:

- 포탑 속사
- 드론 소환
- 과충전 포탑
- 설치물 보호막

플레이 감각:

- 방 안에 작은 요새를 만들고 적을 녹인다.

### 아크 메이지 + 번개 지팡이

초기 방향:

- 상태이상/연쇄 피해 빌드

잘 맞는 증강:

- 감전 전이
- 번개 폭발
- 냉기장
- 상태이상 증폭

플레이 감각:

- 한 마리에게 붙은 상태이상이 방 전체로 번져간다.

## 8. 방 보상 구조

방 클리어 후 보상은 매번 같지 않다.

보상 유형:

- 증강 보드
- 추가 증강 선택권
- 체력 회복
- 상점
- 무기 강화
- 캐릭터 패시브 강화
- 희귀 증강 보드
- 중간보스 보상
- 보스 보상

한 스테이지 안에서 증강을 고르는 횟수는 최소 10회, 목표 12-14회로 잡는다. 중간보스나 이벤트 방을 잘 활용한 경우에는 15회 이상도 가능하다.

초기 프로토타입에서는 모든 방 클리어 후 증강 보드를 보여줘도 된다. 정식 구조에서는 방 타입을 나눠 15분 동안 리듬을 만든다.

## 9. 15분 스테이지 페이스

한 스테이지는 약 15분 플레이를 목표로 한다. 방 하나는 평균 40-60초 전투, 보상 선택은 평균 10-20초를 목표로 잡는다.

권장 방 구성:

| 방 | 타입 | 역할 | 보상 |
| --- | --- | --- | --- |
| 1 | 일반 | 캐릭터/무기 감각 확인 | 일반 증강 |
| 2 | 일반 | 첫 빌드 방향 선택 | 일반 증강 |
| 3 | 일반/특수 | 첫 시너지 후보 제시 | 일반 증강 또는 회복 |
| 4 | 엘리트 | 빌드 약점 확인 | 희귀 확률 증가 |
| 5 | 중간보스 1 | 1차 실력/빌드 체크 | 희귀 3지선다 또는 2회 선택 |
| 6 | 일반 | 중간보스 보상 테스트 | 일반 증강 |
| 7 | 특수/상점 | 리롤, 회복, 강화 선택 | 선택형 보상 |
| 8 | 엘리트 | 적 조합 압박 | 일반/희귀 증강 |
| 9 | 일반 | 빌드 확장 | 일반 증강 |
| 10 | 중간보스 2 | 2차 빌드 체크 | 희귀 3지선다 2회 또는 무기/캐릭터 강화 |
| 11 | 일반 | 완성 빌드 가속 | 일반 증강 |
| 12 | 엘리트 | 고위험 패턴 | 희귀 확률 증가 |
| 13 | 특수 | 회복 또는 강화 분기 | 선택형 보상 |
| 14 | 일반/엘리트 | 최종 보스 전 압박 | 마지막 증강 |
| 15 | 최종 보스 | 스테이지 결산 | 클리어 보상 |

선택지 횟수 기준:

- 최소 보장: 10회
- 권장 목표: 12-14회
- 중간보스 보너스 포함 최대 목표: 15-17회

선택 횟수가 많아지는 대신, 모든 보상이 같은 무게일 필요는 없다.

- 일반 방: 3개 후보 중 1개
- 엘리트 방: 3개 후보 중 1개, 희귀도 가중치 증가
- 중간보스 방: 3개 후보 중 1개를 2회 선택하거나, 3개 후보 모두 희귀도 가중치 증가
- 특수 방: 회복/상점/강화/증강 중 선택
- 최종 보스: 현재 스테이지를 더 강하게 만드는 보상보다 장기 성장/해금 보상 중심

## 10. 방 타입

### 일반 방

- 기본 전투
- 클리어 후 일반 증강 보드

### 엘리트 방

- 강한 적 조합
- 클리어 후 희귀 증강 확률 증가

### 상점 방

- 전투 없음
- 리롤권, 회복, 무기 강화 구매

### 휴식 방

- 체력 회복 또는 보유 증강 강화

### 중간보스 방

- 스테이지 중간의 큰 전투
- 일반 방보다 강한 전용 패턴 보유
- 클리어 후 보상 선택 횟수 또는 희귀도 증가
- 현재 빌드를 굳히거나 새 축을 섞을 수 있는 시점

### 최종 보스 방

- 스테이지 마지막 방
- 클리어 시 스테이지 클리어 보상, 다음 난이도/챕터 해금, 장기 성장 재화 제공

## 11. 스타일리시 연출 방향

전투가 자동이기 때문에 연출이 특히 중요하다. 플레이어는 조작하지 않는 대신 “내 빌드가 멋지게 돌아가는 장면”을 봐야 한다.

연출 원칙:

- 캐릭터 이동은 미끄러지듯 부드럽게
- 대시는 잔상과 화면 흔들림
- 강한 공격은 히트스톱 짧게 적용
- 연쇄 피해는 선명한 라인으로 연결
- 장판은 바닥에 그래픽 패턴을 남김
- 적 처치 시 작지만 날카로운 폭발
- 보스 등장 시 카메라 줌/경고 연출
- 방 클리어 시 짧은 슬로모션과 결과 배너

참고 감각:

- 하데스식 방 클리어 템포
- 궁수의 전설식 방 단위 전투
- 롤 아레나식 과장된 증강 효과
- 롤체식 선택지 조합 읽기

## 12. 현재 프로토타입과의 차이

현재 프로토타입은 아직 이전 방향의 흔적이 있다.

현재 구현:

- 연속 웨이브 생존형에 가까움
- 수동 드래그 이동 개입 가능
