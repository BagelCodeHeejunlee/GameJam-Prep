# 게임잼 기획 리셋

## 목적

이 문서들은 구현을 다시 시작하기 전에 게임 기획을 처음부터 다시 쌓기 위한 작업 공간이다.

현재 방향은 헥사 타일 맵 위에서 캐릭터들이 돌아다니며, 각자의 덱에서 뽑은 카드에 따라 자동으로 행동하는 오토배틀 RPG다.

## 현재 작업 원칙

- 이전 프로토타입 방향은 폐기한 것으로 본다.
- 핵심 콘셉트가 테스트 가능한 수준으로 정리되기 전까지 구현을 우선하지 않는다.
- 기획 결정은 짧고 명확하게 남긴다.
- 기능량보다 작은 플레이 가능 범위를 우선한다.

## 우선 기획 순서

1. 플레이어가 어떤 경험을 하게 할지 정한다.
2. 핵심 전투 루프를 정리한다.
3. 가장 작은 플레이 가능 프로토타입 범위를 잡는다.
4. 첫 빌드에서 검증해야 할 재미를 정한다.
5. 그 검증에 필요 없는 요소는 잘라낸다.

## 열린 질문

- 플레이어는 스테이지 입장 전에 어디까지 준비하고 통제하는가?
- 웨이브 사이에 플레이어가 내리는 결정이 있는가?
- 다음 카드나 적의 행동을 얼마나 보여줄 것인가?
- 캐릭터별 덱은 어떤 방식으로 서로 다르게 느껴지는가?
- 게임잼 기간 안에 빠르게 만들고 테스트할 수 있는 최소 범위는 무엇인가?
