# 몬스터 기획

## 공통 규칙

몬스터도 플레이어 캐릭터와 같은 카드 규칙을 사용한다.

같은 종류의 몬스터는 하나의 공유 덱에서 카드 한 장을 뽑고, 살아있는 같은 종류의 몬스터가 그 카드를 함께 사용한다.

서로 다른 몬스터 종류는 각각 독립적으로 카드를 뽑는다.

현재 프로토타입에는 브루트, 척후병, 사수 3종을 사용한다.

## 브루트

### 역할

기본 근접 몬스터다.

체력이 높고 단순하게 접근해서 공격한다.

### 기본 스탯

| 스탯 | 값 |
| --- | ---: |
| 기본 체력 | 웨이브별 지정 |
| 기본 공격력 | 3 |
| 기본 사거리 | 1 |
| 기본 이동력 | 2 |

### 덱

| 카드 | 장수 | 액션 |
| --- | ---: | --- |
| 접근 공격 | 3 | 이동 2 -> 근거리 공격 1 |
| 공격 후 접근 | 2 | 근거리 공격 1 -> 이동 1 |
| 강타 | 2 | 근거리 공격 2 |
| 접근 | 1 | 이동 2 |

## 척후병

### 역할

빠르게 붙고 빠지는 근접 몬스터다.

브루트보다 체력은 낮지만 우선권이 빠르고, 후퇴 카드를 통해 위치가 자주 바뀐다.

### 기본 스탯

| 스탯 | 값 |
| --- | ---: |
| 기본 체력 | 웨이브별 지정 |
| 기본 공격력 | 3 |
| 기본 사거리 | 1 |
| 기본 이동력 | 3 |

### 덱

| 카드 | 장수 | 액션 |
| --- | ---: | --- |
| 파고들기 | 3 | 이동 3 -> 근거리 공격 1 |
| 치고 빠지기 | 2 | 근거리 공격 1 -> 후퇴 2 |
| 교란 | 2 | 이동 2 |
| 찌르기 | 1 | 근거리 공격 1 |

## 사수

### 역할

원거리 몬스터다.

멀리서 견제하고, 플레이어가 접근하면 공격 후 후퇴한다.

### 기본 스탯

| 스탯 | 값 |
| --- | ---: |
| 기본 체력 | 웨이브별 지정 |
| 기본 공격력 | 4 |
| 기본 사거리 | 3 |
| 기본 이동력 | 2 |

### 덱

| 카드 | 장수 | 액션 |
| --- | ---: | --- |
| 견제 사격 | 3 | 공격 1, 사거리 3 |
| 거리 유지 | 2 | 공격 1, 사거리 3 -> 후퇴 2 |
| 조준 사격 | 1 | 공격 2, 사거리 3 |
| 사격 위치 | 2 | 이동 2 -> 공격 1, 사거리 3 |
