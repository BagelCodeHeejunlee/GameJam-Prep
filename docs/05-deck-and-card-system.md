# 덱과 카드 시스템

## 덱 소유

모든 캐릭터는 각자의 덱을 가진다.

이 규칙은 플레이어 캐릭터와 적 모두에게 적용된다.

## 턴 드로우 규칙

매 턴 관련 덱에서 카드 한 장을 뽑는다.

뽑힌 카드는 해당 캐릭터가 이번 턴에 어떻게 행동할지를 결정한다.

게임이 시작되고 매 턴이 시작되면, 각 캐릭터가 덱에서 카드를 뽑는 연출이 나온다.

적의 경우 같은 종류의 적은 공유 덱 규칙에 따라 한 장만 뽑는다.

적 덱도 덱이 모두 떨어지면 버린카드 더미를 다시 섞어서 사용한다.

## 적 공유 덱 규칙

같은 종류의 적 캐릭터는 하나의 덱 드로우를 공유한다.

즉, 같은 종류의 적들이 각자 카드를 뽑는 것이 아니라 해당 적 종류의 덱에서 카드 한 장을 뽑고, 살아있는 같은 종류의 적들이 그 카드를 함께 사용한다.

예시:

- 살아있는 임프가 2마리 있다.
- 임프는 임프 덱을 사용한다.
- 임프 턴에 임프 덱에서 카드 한 장을 뽑는다.
- 살아있는 임프 2마리는 이번 턴에 같은 카드를 사용한다.

아직 정하지 않은 내용:

- 서로 다른 적 종류는 각각 독립적으로 드로우하는지.
- 죽은 적이 공유 드로우에 영향을 주는지.
- 엘리트나 보스 변종은 별도 덱을 쓰는지.

## 카드 구조

카드는 크게 네 부분으로 구성된다.

| 항목 | 의미 |
| --- | --- |
| 이름 | 카드의 이름. |
| 타입 | 기본 카드, 지속 카드, 영구 카드 중 하나. |
| 액션 | 캐릭터가 이번 턴에 해야 하는 행동. |
| 우선권 | 카드의 플레이 순서. 낮은 우선권 수치가 먼저 플레이된다. |

## 카드 타입

카드는 총 3가지 타입으로 나눌 수 있다.

### 기본 카드

이동이나 공격 등 평범한 액션이 있는 카드다.

사용 후 일반적인 버린카드 더미 처리 규칙을 따른다.

### 지속 카드

다음 내 턴이 끝날 때까지 특수한 효과를 주는 카드다.

예시:

- 반격.
- 공격 방어.

지속 카드는 다음 내 턴이 끝날 때까지 영향을 준다.

내 턴이 끝날 때 버린카드 더미로 이동한다.

### 영구 카드

이번 웨이브를 클리어할 때까지 계속 나에게 영향을 주는 카드다.

예시:

- 특수한 소환.
- 점프 시 공격.

영구 카드는 웨이브가 끝날 때까지 버린카드 더미로 들어가지 않는다.

따라서 한 번 사용하면 해당 웨이브 중에는 다시 뽑을 수 없다.

웨이브 종료 처리와 다음 웨이브 시작 준비 단계에서 영구 카드는 다시 덱으로 돌아온다.

이때 버린카드 더미에 있던 카드들도 모두 다시 덱으로 들어온다.

## 카드 액션

액션은 캐릭터가 이번 턴에 무엇을 하는지 나타낸다.

카드의 액션부는 항상 위에서 아래 순서로 실행된다.

예시:

```text
공격 2
이동 3
```

위 카드라면 기본 공격력의 2배로 먼저 공격하고, 그 다음 3칸 이동한다.

```text
이동 3
공격 2
```

위 카드라면 먼저 3칸 이동하고, 그 다음 기본 공격력의 2배로 공격한다.

현재 정리된 액션 관련 정보:

- 카드에 따라 공격 사거리가 달라질 수 있다.
- 카드에서 나온 배수를 기본 공격력에 곱해서 사용한다.
- 사거리는 이동 기준으로도 사용되므로, 카드 액션은 이동에도 영향을 줄 수 있다.

## 카드 우선권

우선권은 카드의 플레이 순서를 결정한다.

우선권 수치가 낮은 카드가 먼저 플레이된다.

우선권 숫자 범위는 1에서 99 사이다.

우선권이 같은 경우 플레이어와 몬스터 사이에서는 플레이어가 먼저 행동한다.

몬스터끼리 우선권이 같거나 플레이어 캐릭터끼리 우선권이 같다면, 아무 순서로 먼저 행동해도 된다.

매 턴 뽑힌 카드는 전투 HUD를 유지한 상태에서 간단히 보여주고, 우선권 순서대로 정리되는 연출을 보여준다.

카드 공개 연출은 별도의 새 화면처럼 보이지 않게 한다.

정렬이 끝나면 같은 우선권 HUD의 토큰들이 그대로 현재 턴 실행 순서 표시 역할을 이어받는다.

프로토타입 카드 UI 기준:

- 카드 왼쪽 위에는 동그라미 안에 우선권 숫자를 표시한다.
- 카드 위쪽 중앙에는 카드 이름을 표시한다.
- 액션부는 카드 중앙에 배치한다.
- 여러 액션이 이어지는 카드는 액션 사이에 얇은 가로 구분선을 둔다.
- 이동, 근접 공격, 원거리 공격, 사거리, 타겟 수는 텍스트보다 아이콘과 숫자로 보여준다.
- 근접 공격은 별도 사거리 숫자를 표시하지 않아도 된다.
- 타일 모양 공격은 카드 안에 공격 타일 모양을 함께 표시한다.

프로토타입 전투 HUD 기준:

- 상단에는 적 종류별 정보를 한 줄로 구분해서 보여준다.
- 적 정보에는 적 초상/라벨과 수량을 우선 보여준다. 공간이 충분할 때만 기본 이동력, 기본 공격력, 기본 사거리를 함께 보여준다.
- 적 쪽에는 매 턴 카드 전체를 상시 표시하지 않는다.
- 적 정보 아래 한 줄에는 현재 턴 우선권 순서를 작게 보여준다.
- 우선권 순서는 카드 이름보다 우선권 숫자와 행동 주체 식별을 우선한다.
- 하단에는 플레이어 캐릭터 정보를 표시한다.
- 플레이어 캐릭터가 1명일 때는 같은 크기의 HUD를 중앙에 배치하고, 2명일 때는 좌우로 분리한다.
- 플레이어 HUD에는 캐릭터 초상/라벨, 기본 이동력, 기본 공격력, 기본 사거리, 덱/버린 더미/차지 수치, 이번 턴 뽑은 카드 미리보기를 보여준다.

아직 정하지 않은 내용:

- 지속 카드와 영구 카드가 액션 실행 후 어느 영역에 놓이는지에 대한 UI 표현.
