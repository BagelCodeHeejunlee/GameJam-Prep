# 보상과 빌드 루트

## 웨이브 클리어 보상

웨이브를 클리어하면 플레이어 캐릭터의 보상 카드 풀에서 카드 4장을 보여준다.

플레이어는 이 4장 중 1장을 선택해야 한다.

현재 프로토타입에서는 보상 선택지를 2x2 배열로 보여준다.

선택한 카드는 캐릭터의 덱에 들어오며, 이번 스테이지 동안 사용할 수 있다.

선택하지 않은 카드도 이후 보상 선택지에 다시 등장할 수 있다.

이미 가진 카드도 다시 등장할 수 있다.

이 재등장 여부와 최대 등장 가능 장수는 카드 데이터에 값으로 둔다.

같은 카드가 선택지로 나왔다면 선택할 수 있다.

보상 선택창은 기본적으로 카드만 보여준다.

카드 안에는 제목, 우선권, 액션 아이콘, 선택 표시를 우선 노출한다.

등급 텍스트와 루트 텍스트는 카드 안에서 직접 표시하지 않아도 된다.

현재 프로토타입에서는 카드의 등급을 카드 배경 이미지와 색감으로 구분한다.

카드 우상단의 설명 버튼을 누르면 각 아이콘의 설명이 나오는 창이 열린다.

## 캐릭터별 보상 카드 풀

각 캐릭터는 자기만의 보상 카드 풀을 가진다.

보상 카드 풀은 해당 캐릭터가 어떤 방향으로 빌드될 수 있는지 결정한다.

카드 풀은 충분한 수량으로 준비되어야 한다.

기준 수량:

- 각 루트별 약 20장.
- 한 캐릭터가 평균 3개 루트를 가진다면 약 60장.

## 빌드 루트 구조

각 캐릭터는 최소 2개, 평균 3개 정도의 빌드 루트를 가져야 한다.

이 루트는 선형적인 레벨업 트리가 아니다.

플레이어가 웨이브 클리어 보상으로 어떤 카드를 선택하느냐에 따라 조합으로 만들어지는 방향성이다.

즉, 캐릭터가 정해진 순서대로 강화되는 것이 아니라, 선택한 카드들이 쌓이면서 이번 스테이지의 빌드가 형성된다.

## 카드 등급

보상 카드는 등급을 가진다.

현재 등급:

| 등급 | 카드 배경 방향 |
| --- | --- |
| 노말 | 회색 |
| 레어 | 파란색 |
| 에픽 | 보라색 |
| 전설 | 노란색 또는 금색 |

등급별 등장 확률은 추후 결정한다.

## 루트별 카드 풀 기준

각 루트는 대략 다음 비율의 카드 풀을 가진다.

| 등급 | 장수 |
| --- | ---: |
| 노말 | 8장 |
| 레어 | 5장 |
| 에픽 | 5장 |
| 전설 | 3장 |
| 합계 | 21장 |

루트별 카드 풀은 약 20장 기준으로 잡는다.

## 궁수 캐릭터 루트 예시

궁수 캐릭터는 다른 캐릭터와 마찬가지로 8장의 기본 카드로 시작한다.

웨이브 클리어 보상으로 등장하는 카드 선택에 따라 다음과 같은 루트로 분기될 수 있다.

### 다단중첩

공격을 맞을수록 피해가 증가하는 루트.

핵심 방향:

- 동일 타겟을 반복 공격한다.
- 같은 적을 여러 번 공격할수록 피해량이나 부가 효과가 증가한다.
- 타겟 집중과 다단 히트가 핵심 재미가 된다.

### 함정

함정 설치와 밀기로 함정 효과를 트리거하는 루트.

핵심 방향:

- 헥사 타일에 함정을 만든다.
- 적이 지나갈 위험 구역을 만든다.
- 적을 밀어서 함정 효과를 발동시킨다.

### 차지

공격을 차지해서 강한 한 방을 만드는 루트.

핵심 방향:

- 이동을 포기하거나 제한하면서 차지를 쌓는다.
- 쌓은 차지를 강력한 공격으로 전환한다.
- 적은 공격 횟수로 큰 피해를 주는 것이 핵심 재미가 된다.

## 아직 정하지 않은 내용

- 여러 루트를 섞는 하이브리드 빌드를 의도적으로 지원할지.
- 캐릭터별 기본 카드도 루트와 연결되는지, 완전히 범용인지.
