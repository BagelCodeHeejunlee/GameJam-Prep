# 기본 선택지 풀

## 1. 목적

이 문서는 모든 캐릭터가 공통으로 사용할 수 있는 기본 선택지 풀을 정의한다. 캐릭터별 선택지는 별도 풀로 관리한다.

기본 선택지는 특정 캐릭터 전용 능력이 아니라, 여러 캐릭터와 무기가 서로 다른 방식으로 활용할 수 있는 빌드 재료여야 한다.

기준:

- 모든 보상 화면은 3개 후보 중 1개 선택
- 후보 수는 늘리지 않는다
- 큰 보상은 추가 선택권, 희귀도 가중치, 보상 품질 상승으로 표현
- 선택지는 노말, 레어, 에픽, 전설 4등급으로 구분
- 유저 레벨이 높아질수록 더 많은 기본 선택지가 해금
- 캐릭터 레벨이 높아질수록 해당 캐릭터 전용 선택지가 해금
- 기본 풀에는 특정 캐릭터만 사용할 수 있는 소환, 설치, 코어 전용 선택지를 넣지 않는다

연쇄 선택지 등장 조건:

- `장판 확장`, `중력장`, `장판 처형`은 `플라즈마 잔상`, `냉기장`, `대시 지뢰`, `무한 궤도` 중 하나를 이미 보유했을 때만 등장한다.
- `특이점 붕괴`는 장판 시작점과 `중력장`을 모두 보유했을 때만 등장한다.
- `대시 지뢰`, `대시 보호막`은 `자동 대시`를 이미 보유했을 때만 등장한다.
- `냉기 파열`은 냉기 부여 수단을 보유했을 때만 등장한다.
- `감전 전이`, `번개 폭발`은 감전 부여 수단을 보유했을 때만 등장한다.

구현 규격:

```js
repeatable: true
max: 3
requires: [{ anyTags: ["장판"] }]
requires: [{ anyIds: ["autoDash"] }]
requires: [{ anyTags: ["장판"] }, { anyIds: ["gravity"] }]
```

- `repeatable: true`인 선택지만 이미 선택한 뒤에도 다시 후보에 등장할 수 있다.
- `repeatable: false`인 선택지는 한 번 선택하면 이후 보상 후보에서 제외된다.
- `max`는 반복 가능 선택지의 최대 중첩 수다.
- `requires` 배열 안의 조건들은 모두 만족해야 한다.
- 한 조건 안의 `anyIds`, `anyTags`는 하나만 만족해도 통과한다.
- 사전 조건을 만족하지 못한 선택지는 보상 후보 3개에 들어가지 않는다.

반복 가능 여부 기준:

- 기능 해금형 선택지는 `repeatable: false`로 둔다. 예: `플라즈마 잔상`, `냉기장`, `대시 지뢰`, `자동 대시`, `표식 탄환`, `장판 처형`.
- 수치 성장형 선택지는 `repeatable: true`로 둔다. 예: `고압 탄환`, `속사 모듈`, `관통 탄환`, `장판 확장`, `대시 보호막`, `전술 스캐너`.
- 기능 해금형을 강화하고 싶다면 같은 선택지를 반복시키지 말고 별도 강화 선택지를 만든다. 예: `잔상 과열`, `냉기장 증폭`, `지뢰 연쇄`.

## 7. 대표 연계 시너지

### 탄환 성장

`고압 탄환 -> 쌍열 발사 -> 삼연 발사 -> 탄막 과부하`

탄환 수와 발동 횟수를 늘려 모든 탄환 기반 효과의 가치를 올린다.

### 냉기 제어

`냉기 탄환 -> 냉기장 -> 냉기 파열 -> 폭풍 전도체`

적을 느리게 만들고, 처치 시 냉기 폭발로 방 전체를 제어한다.

### 감전 연쇄

`감전 탄환 -> 감전 전이 -> 속사 모듈 -> 폭풍 전도체`

빠른 공격으로 감전을 퍼뜨리고, 상태 전이를 방 전체 피해로 확장한다.

### 이동 장판

`기동 부츠 -> 자동 대시 -> 플라즈마 잔상 -> 장판 처형 -> 무한 궤도`

자동 이동 경로 자체가 공격 수단이 된다.

### 표식 집중

`표식 탄환 -> 표식 사냥 -> 속사 모듈 -> 탄막 과부하`

한 대상에게 공격을 집중할수록 강해져 중간보스와 최종 보스에 강해진다.

### 중력 폭발

`플라즈마 잔상 -> 중력장 -> 장판 처형 -> 특이점 붕괴`

적을 장판 위로 모으고, 처형과 폭발로 방 전체를 정리한다.

### 드래프트 경제

`데이터 자석 -> 전술 스캐너 -> 리롤 장치 -> 전술 지배`

당장 전투력은 낮지만 이후 보상 품질과 시너지 후보를 조정한다.

## 8. 프로토타입 기준

프로토타입에서는 위 33개 기본 선택지만 먼저 구현한다.

캐릭터 전용 선택지는 다음 단계에서 각 캐릭터당 아래 수량으로 확장한다.

- 노말 3개
- 레어 3개
- 에픽 2개
- 전설 1개

캐릭터 전용 선택지는 캐릭터의 자동 이동 성향과 고유 패시브를 더 강하게 비트는 방향으로 설계한다.
