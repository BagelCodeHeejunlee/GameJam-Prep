# 헥스 자동 전투 로그라이크 프로토타입 기획서

## 1. 핵심 방향

기존 헥스 전투의 공간감은 유지한다. 다만 전투 중 카드 드로우, 카드 순서 선택, 직접 타겟팅은 제거한다.

새 구조는 **3명의 영웅이 각자 고정 행동 루틴을 자동 실행하고, 유저는 웨이브 후 로그라이크 선택지로 루틴을 업그레이드하는 게임**이다.

핵심 재미:

> 내가 고른 업그레이드 때문에 궁수/전사/마법사의 자동 행동이 점점 조합처럼 굴러가는 것을 보는 재미.

## 2. 보존 원칙

- 루트 `app.js`의 기존 카드/보상 데이터는 삭제하지 않는다.
- 기존 카드 시스템으로 되돌아갈 수 있게 새 프로토타입은 별도 폴더에서 구현한다.
- 궁수/전사/마법사의 기존 트리 방향은 유지한다.

## 3. 전투 입력

전투 중 유저 입력은 없다.

유저가 하지 않는 것:

- 카드 선택
- 카드 순서 선택
- 타겟 선택
- 이동 위치 선택
- 우선도 계산

유저가 하는 것:

- 웨이브 클리어 후 선택지 3개 중 1개 선택

## 4. 기본 자동 루틴

### 궁수

기존 방향: `함정`, `연타`, `표식/저격`

기본 루틴:

1. 안전 거리를 유지하도록 최대 2칸 이동
2. 사거리 안의 위험한 적에게 원거리 공격 1회

### 전사

기존 방향: `광전`, `돌진`, `범위 공격`

기본 루틴:

1. 가장 가까운 적에게 최대 3칸 접근
2. 근접 공격 1회

### 마법사

기존 방향: `연쇄`, `룬`, `운석`

기본 루틴:

1. 적과 너무 가까우면 최대 1칸 후퇴
2. 사거리 안의 밀집/위험 적에게 마법 공격 1회

## 5. 선택지 설계 원칙

선택지는 카드가 아니라 **행동 루틴 업그레이드**다.

좋은 선택지는 다음 중 하나를 바꿔야 한다.

- 이동 방식
- 공격 횟수
- 타겟 우선순위
- 공격 후 추가 효과
- 주기적 자동 스킬
- 다른 영웅 행동과의 연동

## 6. 캐릭터별 선택지

### 궁수

| 선택지 | 트리 | 효과 |
| --- | --- | --- |
| 안정 사격 | 공용 | 궁수 기본 피해 +1 |
| 사거리 확보 | 공용 | 궁수 사거리 +1 |
| 약점 표식 | 표식 | 궁수 공격이 표식 1 부여 |
| 표식 사냥 | 표식 | 표식 적 공격 피해 +2 |
| 표식 전이 | 표식 | 표식 적 처치 시 다른 적에게 표식 이전 |
| 반복 리듬 | 연타 | 같은 적을 다시 공격하면 추가 피해 |
| 연속 사격 | 연타 | 궁수가 추가 공격 1회 |
| 삼연사 | 연타 | 궁수 추가 공격 2회 |
| 공격 함정 | 함정 | 궁수가 주기적으로 함정 설치 |
| 함정 손질 | 함정 | 함정 피해 증가 |
| 가시 길목 | 함정 | 함정이 적 이동력을 낮춤 |
| 연쇄 함정 | 함정 | 함정 발동 시 주변 피해 |

### 전사

| 선택지 | 트리 | 효과 |
| --- | --- | --- |
| 강한 베기 | 공용 | 전사 기본 피해 +1 |
| 전열 유지 | 공용 | 전사가 낮은 체력 아군 주변 적을 우선 처리 |
| 방패 밀치기 | 돌진 | 전사 공격이 적을 1칸 밀침 |
| 돌파 | 돌진 | 이동 후 공격 피해 증가 |
| 전장의 혜성 | 돌진 | 3칸 이상 이동하면 추가 피해와 밀침 |
| 넓은 휩쓸기 | 범위 공격 | 전사 공격이 인접 적에게도 피해 |
| 전열 붕괴 | 범위 공격 | 밀린 적 주변에 피해 |
| 전장 장악 | 범위 공격 | 범위 피해 증가 |
| 피의 박자 | 광전 | 전사 체력이 낮을수록 피해 증가 |
| 광전 태세 | 광전 | 전사 공격 주기 감소 |
| 피의 폭풍 | 광전 | 일정 턴마다 주변 다중 타격 |

### 마법사

| 선택지 | 트리 | 효과 |
| --- | --- | --- |
| 안정된 마력 | 공용 | 마법사 기본 피해 +1 |
| 마력 거리두기 | 공용 | 마법사 후퇴 루틴 강화 |
| 연쇄 마력탄 | 연쇄 | 마법 공격이 다른 적에게 전이 |
| 세 갈래 번개 | 연쇄 | 전이 횟수 증가 |
| 무한 회로 | 연쇄 | 전이 피해 증가 |
| 폭발 룬 | 룬 | 마법사가 주기적으로 룬 설치 |
| 룬 조율 | 룬 | 룬 피해 증가 |
| 룬 연동 | 룬 | 룬 발동 시 주변 룬도 발동 |
| 운석 예고 | 운석 | 일정 턴마다 운석 낙하 |
| 파편 회수 | 운석 | 운석 처치 시 다음 운석 앞당김 |
| 하늘 붕괴 | 운석 | 운석 범위와 피해 증가 |

## 7. 파티 시너지

| 선택지 | 방향 | 효과 |
| --- | --- | --- |
| 표식 돌파 | 궁수 + 전사 | 전사가 표식 적을 공격하면 밀침과 추가 피해 |
| 표식 전도 | 궁수 + 마법사 | 마법 연쇄가 표식을 복사 |
| 불타는 함정 | 궁수 + 마법사 | 함정 발동 시 화상 부여 |
| 충돌 폭발 | 전사 + 마법사 | 전사가 밀친 적 주변에 마법 피해 |
| 룬 돌진 | 전사 + 마법사 | 전사가 밀친 위치에 룬 생성 |
| 포위 완성 | 3인 | 표식 적이 밀린 뒤 마법 피해를 받으면 폭발 |

## 8. 프로토타입 성공 기준

1. 유저가 전투 중 조작하지 않아도 전투가 읽히는가?
2. 웨이브 후 선택지가 다음 전투에서 눈에 띄게 보이는가?
3. 3명의 영웅이 모두 자기 역할을 한다고 느껴지는가?
4. 궁수/전사/마법사의 기존 트리 정체성이 유지되는가?
5. 카드 시스템 없이도 로그라이크 성장 선택이 매력적인가?
